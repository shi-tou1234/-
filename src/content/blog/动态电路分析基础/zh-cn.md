## 一、 动态电路的基本概念与输入输出方程

### 1. 核心概念
- **动态元件**：电容（$C$）与电感（$L$）。电容需变化的电压才能产生电流，电感需变化的电流才能产生电压。
- **动态电路**：包含动态元件的电路。
- **稳态与暂态**：在恒定激励下，电路电压电流保持恒定的工作状态称为**稳态**。当电路发生**换路**（如开关通断、电源突变、参数跳变等）时，由于储能元件的能量无法瞬间改变，电路会经历一个过渡过程，称为**暂态过程**。暂态结束后，电路进入新的稳态。
- **输入与输出**：激励（独立电压源或电流源）称为**输入**，待求的电压或电流响应称为**输出**。仅含一个激励和一个输出变量的电路称为**单输入-单输出电路**。

### 2. 输入输出方程的列写
动态电路的输入输出方程本质上是线性常系数微分方程。列写依据为基尔霍夫定律（KCL/KVL）与元件的电压电流关系（VCR）。

**例1：RLC串联电路（以 $u_C(t)$ 为输出）**
电路由 $R, L, C$ 与电压源 $u_S(t)$ 串联。取顺时针回路，由KVL得：
$$ u_R(t) + u_L(t) + u_C(t) = u_S(t) $$
根据元件VCR：
$$ i(t) = C \frac{\mathrm{d}u_C(t)}{\mathrm{d}t} $$
$$ u_L(t) = L \frac{\mathrm{d}i(t)}{\mathrm{d}t} = L \frac{\mathrm{d}}{\mathrm{d}t} \left( C \frac{\mathrm{d}u_C(t)}{\mathrm{d}t} \right) = LC \frac{\mathrm{d}^2 u_C(t)}{\mathrm{d}t^2} $$
$$ u_R(t) = R i(t) = RC \frac{\mathrm{d}u_C(t)}{\mathrm{d}t} $$
将上述表达式代入KVL方程，整理得关于 $u_C(t)$ 的二阶微分方程：
$$ LC \frac{\mathrm{d}^2 u_C(t)}{\mathrm{d}t^2} + RC \frac{\mathrm{d}u_C(t)}{\mathrm{d}t} + u_C(t) = u_S(t) $$
化为标准形式：
$$ \frac{\mathrm{d}^2 u_C(t)}{\mathrm{d}t^2} + \frac{R}{L} \frac{\mathrm{d}u_C(t)}{\mathrm{d}t} + \frac{1}{LC} u_C(t) = \frac{1}{LC} u_S(t) $$

**例2：RLC并联电路（以 $i_L(t)$ 和 $u(t)$ 为输出）**
电路由电流源 $i_S(t)$ 与 $R, L, C$ 并联。由KCL得：$i_S = i_C + i_R + i_L$。
- **以 $i_L$ 为输出**：
  并联电压 $u = L \frac{\mathrm{d}i_L}{\mathrm{d}t}$，则 $i_C = C \frac{\mathrm{d}u}{\mathrm{d}t} = LC \frac{\mathrm{d}^2 i_L}{\mathrm{d}t^2}$，$i_R = \frac{u}{R} = \frac{L}{R} \frac{\mathrm{d}i_L}{\mathrm{d}t}$。
  代入KCL方程得：
  $$ LC \frac{\mathrm{d}^2 i_L}{\mathrm{d}t^2} + \frac{L}{R} \frac{\mathrm{d}i_L}{\mathrm{d}t} + i_L = i_S(t) $$
- **以 $u$ 为输出**：
  $i_L = \frac{1}{L} \int_{-\infty}^{t} u(\tau) \mathrm{d}\tau$，$i_C = C \frac{\mathrm{d}u}{\mathrm{d}t}$，$i_R = \frac{u}{R}$。
  代入KCL得积分微分方程：
  $$ C \frac{\mathrm{d}u}{\mathrm{d}t} + \frac{u}{R} + \frac{1}{L} \int_{-\infty}^{t} u(\tau) \mathrm{d}\tau = i_S(t) $$
  两边对 $t$ 求导，消去积分项，得二阶微分方程：
  $$ C \frac{\mathrm{d}^2 u}{\mathrm{d}t^2} + \frac{1}{R} \frac{\mathrm{d}u}{\mathrm{d}t} + \frac{1}{L} u = \frac{\mathrm{d}i_S(t)}{\mathrm{d}t} $$

### 3. 电路阶数
输入输出方程为 $n$ 阶微分方程的电路称为 **$n$ 阶电路**。通常，电路中独立储能元件的个数决定了方程的阶数。在列写方程时，以电容电压 $u_C$ 或电感电流 $i_L$ 为变量最为简便。

## 二、 换路定则与初始条件的计算

### 1. 换路瞬间的数学描述
换路通常定义在 $t=0$ 时刻。为精确描述换路前后的转折瞬间，引入：
- $t=0^-$：换路前的瞬时（极限趋近于0）。
- $t=0^+$：换路后的初始瞬时（极限趋近于0）。
动态电路分析的核心是求解 $t=0^+$ 时刻各变量的**初始值**。

### 2. 换路定则（Switching Rule）
若换路瞬间电容电流 $i_C$ 和电感电压 $u_L$ 为有限值（非冲激信号），则电容电压和电感电流**不能发生跃变**。

**严格推导**：
由电容的积分关系 $q_C(t) = q_C(t_m) + \int_{t_m}^{t} i_C(\tau) \mathrm{d}\tau$，令 $t_m=0^-$，$t=0^+$：
$$ q_C(0^+) = q_C(0^-) + \int_{0^-}^{0^+} i_C(\tau) \mathrm{d}\tau $$
当 $i_C(t)$ 有界时，积分区间趋于零，积分值为0，故 $q_C(0^+) = q_C(0^-)$。结合 $q_C = C u_C$，得：
$$ u_C(0^+) = u_C(0^-) $$
同理，由电感的积分关系 $\Psi_L(t) = \Psi_L(t_m) + \int_{t_m}^{t} u_L(\tau) \mathrm{d}\tau$，令 $t_m=0^-$，$t=0^+$：
$$ \Psi_L(0^+) = \Psi_L(0^-) + \int_{0^-}^{0^+} u_L(\tau) \mathrm{d}\tau = \Psi_L(0^-) $$
结合 $\Psi_L = L i_L$，得：
$$ i_L(0^+) = i_L(0^-) $$

**结论**：
$$ u_C(0^+) = u_C(0^-), \quad i_L(0^+) = i_L(0^-) $$
> 注：电阻电压/电流、电容电流、电感电压等其他变量在换路瞬间**可能发生跃变**。

### 3. 初始值的系统化计算步骤
1. **求 $t=0^-$ 原始值**：换路前电路已达直流稳态。此时电容视为开路（$i_C=0$），电感视为短路（$u_L=0$）。利用电阻电路分析法求解，得到 $u_C(0^-)$ 和 $i_L(0^-)$。
2. **应用换路定则**：直接确定 $u_C(0^+) = u_C(0^-)$，$i_L(0^+) = i_L(0^-)$。
3. **构建 $t=0^+$ 等效电路**：
   - 将电容替换为电压值为 $u_C(0^+)$、方向一致的独立电压源。
   - 将电感替换为电流值为 $i_L(0^+)$、方向一致的独立电流源。
   - 独立电源保持换路后的状态。
4. **求解其他初始值**：此时电路为纯电阻电路。应用KCL、KVL、欧姆定律或节点/网孔分析法，求出所需变量在 $t=0^+$ 的初始值。
5. **高阶导数初始值**：$n$ 阶电路需补充输出变量 $1 \sim (n-1)$ 阶导数的初始值。通常将 $t=0^+$ 时刻的各已知值代入原微分方程逐级求解。

## 三、 一阶电路的零输入响应

### 1. 响应的分类
- **零输入响应 (ZIR)**：无独立源激励，仅由储能元件初始储能（$u_C(0^-)\neq 0$ 或 $i_L(0^-)\neq 0$）引起的响应，本质是储能释放过程。
- **零状态响应 (ZSR)**：初始储能为零（$u_C(0^-)=0, i_L(0^-)=0$），仅由独立源激励引起的响应。
- **全响应**：初始储能与独立源共同作用引起的响应。满足线性叠加：全响应 = 零输入响应 + 零状态响应。

### 2. 一阶电路与两要素法
仅含一个独立储能元件（或可等效为单个）的电路为一阶电路。其零输入响应具有统一的指数衰减形式：
$$ r(t) = r(0^+) e^{-\frac{t}{\tau}}, \quad t \ge 0 $$
其中：
- $r(0^+)$：目标变量在 $t=0^+$ 的初始值。
- $\tau$：电路的时间常数。
  - $RC$ 电路：$\tau = R_{eq}C$
  - $RL$ 电路：$\tau = \frac{L}{R_{eq}}$
- $R_{eq}$：从储能元件两端看进去的**无源一端口等效电阻**（需将电路中所有独立源置零：电压源短路，电流源开路，受控源保留）。

**公式推导（以RC串联零输入电路为例）**：
KVL: $u_R + u_C = 0 \Rightarrow R i + u_C = 0$。
代入 $i = C \frac{\mathrm{d}u_C}{\mathrm{d}t}$，得一阶齐次微分方程：
$$ RC \frac{\mathrm{d}u_C}{\mathrm{d}t} + u_C = 0 $$
特征方程为 $RC \lambda + 1 = 0 \Rightarrow \lambda = -\frac{1}{RC}$。
通解为 $u_C(t) = A e^{-\frac{t}{RC}}$。由初始条件 $u_C(0^+) = U_0$ 定积分常数 $A = U_0$。
故 $u_C(t) = U_0 e^{-\frac{t}{RC}}$，时间常数 $\tau = RC$。电流及其他支路电压同理可得。

### 3. 计算步骤（两要素法）
1. **求初始值 $r(0^+)$**：按第二节方法求出 $u_C(0^+)$ 或 $i_L(0^+)$，构建 $t=0^+$ 等效电阻电路，计算目标变量初始值。
2. **求等效电阻 $R_{eq}$ 与时间常数 $\tau$**：
   - 移除储能元件，独立源置零。
   - 若含受控源，采用**加压求流法**：在端口加测试电压 $u$，求流入电流 $i$，则 $R_{eq} = u/i$。需列写KCL/KVL消去控制量。
   - 计算 $\tau = R_{eq}C$ 或 $\tau = L/R_{eq}$（注意单位统一为 $\Omega, F, H$，结果单位为秒）。
3. **代入公式**：直接写出 $r(t) = r(0^+) e^{-t/\tau}$。

**典型应用场景**：
- **RC放电**：$u_C(t) = u_C(0^+) e^{-t/(R_{eq}C)}$，电压按指数规律衰减至0。
- **RL放电**：$i_L(t) = i_L(0^+) e^{-t/(L/R_{eq})}$，电流按指数规律衰减至0。
- 含受控源或复杂电阻网络时，$R_{eq}$ 的计算是核心，务必保留受控源关系进行代数求解。

## 总结
动态电路暂态分析遵循严密的逻辑链条：
1. **列方程**：基于KCL/KVL与VCR建立描述电路行为的微分方程，确定电路阶数。
2. **定初值**：利用直流稳态特性求 $t=0^-$ 值，通过换路定则过渡到 $t=0^+$，构建等效电阻电路求解各变量初始条件。
3. **求响应**：对一阶电路，熟练掌握“两要素法”（初始值+时间常数），利用指数衰减通解快速写出零输入响应；对高阶电路，则需求解特征根并结合初始条件确定积分常数。
掌握上述方法，即可系统、准确地分析各类动态电路的暂态过程。
