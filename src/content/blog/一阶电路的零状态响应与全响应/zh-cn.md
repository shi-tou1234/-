# 一阶电路的零状态响应与全响应

> 本文整理自电路原理课程笔记，详细推导一阶电路零状态响应、阶跃响应及全响应的核心概念与求解方法。

---

## 一、零状态响应

### 1.1 基本概念

**零状态**是零原始状态的简称，指电容电压和电感电流的原始值为 0，即：

$$
u_C(0_-) = 0, \quad i_L(0_-) = 0
$$

此时电容和电感的原始储能为 0。

**零状态响应**：电路在零原始状态下，仅由独立源所引起的响应。

> 根据换路定则，换路后电容电压和电感电流的初始值也为 0：
> $$
> u_C(0_+) = u_C(0_-) = 0, \quad i_L(0_+) = i_L(0_-) = 0
> $$
> 这是求解零状态响应的前提条件。

### 1.2 一阶电路零状态响应的分类

根据输入电源形式不同，零状态响应可分为四类：

| 电源类型 | 响应名称 | 特点 |
|---------|---------|------|
| 恒定直流电源 | 直流零状态响应 | 电源值为常数 |
| 周期性电源（正弦） | 正弦零状态响应 | 电源为正弦函数 |
| 单位阶跃电源 | **单位阶跃响应**（阶跃响应） | 电源值为 $\varepsilon(t)$ |
| 单位冲激电源 | **单位冲激响应**（冲激响应） | 电源值为 $\delta(t)$ |

**阶跃响应**：在单位阶跃电压或电流作用下的零状态响应，满足：
$$
u_C(0_-) = 0, \quad i_L(0_-) = 0
$$

## 二、三要素法求解一阶电路零状态响应

### 2.1 通用公式

当 $t \geq 0_+$ 时，一阶电路的任意零状态响应 $r(t)$ 可表示为：

$$
r(t) = r(\infty) + \left[ r(0_+) - r(\infty) \right] e^{-\frac{t}{\tau}}, \quad t \geq 0
$$

:::derivation
**三要素法公式的推导**：

一阶电路的微分方程的一般形式为（以 DC 激励为例）：

$$\tau \frac{\mathrm{d}r}{\mathrm{d}t} + r = r(\infty)$$

其中 $r(\infty)$ 为稳态值（常数），$\tau$ 为时间常数。该方程为**一阶线性常系数非齐次微分方程**，其通解 = 齐次解 + 特解：

- **齐次解**（对应齐次方程 $\tau \dot{r} + r = 0$）：$r_h(t) = Ae^{-t/\tau}$
- **特解**（激励为常数）：$r_p(t) = r(\infty)$

故通解为：

$$r(t) = r(\infty) + Ae^{-t/\tau}$$

代入初始条件 $r(0_+) = r(\infty) + A$，解得 $A = r(0_+) - r(\infty)$，因此：

$$r(t) = r(\infty) + \left[r(0_+) - r(\infty)\right]e^{-t/\tau}$$

该式表明：一阶电路的任意响应均可由**初始值 $r(0_+)$、稳态值 $r(\infty)$、时间常数 $\tau$** 三个要素唯一确定，故称"三要素法"。
:::

对于阶跃响应，需乘以单位阶跃函数 $\varepsilon(t)$：

$$
r(t) = \left\{ r(\infty) + \left[ r(0_+) - r(\infty) \right] e^{-\frac{t}{\tau}} \right\} \varepsilon(t)
$$

:::derivation
**阶跃响应乘以 $\varepsilon(t)$ 的推导**：

三要素法公式 $r(t) = r(\infty) + [r(0_+) - r(\infty)]e^{-t/\tau}$ 仅对 $t \geq 0$ 成立。对于零状态响应（$t < 0$ 时电路无激励，响应为零），需限定响应的定义域：

$$r(t) = \begin{cases} 0, & t < 0 \\ r(\infty) + [r(0_+) - r(\infty)]e^{-t/\tau}, & t \geq 0 \end{cases}$$

利用单位阶跃函数 $\varepsilon(t)$ 的定义（$t<0$ 时为 0，$t>0$ 时为 1），上式可简洁地写成：

$$r(t) = \left\{r(\infty) + [r(0_+) - r(\infty)]e^{-t/\tau}\right\}\varepsilon(t)$$

乘以 $\varepsilon(t)$ 的物理意义是：保证 $t < 0$ 时响应为零（因果性），同时不影响 $t \geq 0$ 时的响应值。
:::

### 2.2 三要素定义

| 要素 | 符号 | 求解方法 |
|-----|------|---------|
| **初始值** | $r(0_+)$ | 电容电压/电感电流初始值为 0；其他变量通过 $t=0_+$ 等效电阻电路求解 |
| **稳态值** | $r(\infty)$ | $t \to \infty$ 时，电容开路、电感短路，用电阻电路分析方法求解 |
| **时间常数** | $\tau$ | RC 电路：$\tau = R_{eq}C$；RL 电路：$\tau = L/R_{eq}$ |

### 2.3 等效电阻 $R_{eq}$ 的求解

1. 移除电容或电感元件，得到有源一端口电路
2. 将独立源置零（电压源短路、电流源开路）
3. 求无源一端口电路的端口等效电阻

### 2.4 三要素法解题步骤
1.求初始值 r(0+)
电容电压/电感电流：r(0+) = 0（零状态前提）
其他变量：t=0+ 时用 0V 电压源替代电容（短路），0A 电流源替代电感（开路），求解电阻电路

2.求稳态值 r(∞)
t→∞ 时电容开路、电感短路
用电阻电路分析方法（KCL/KVL/节点电压法）求解

3.求时间常数 τ
移除动态元件，独立源置零
求端口等效电阻 Req
τ = Req·C（RC 电路）或 τ = L/Req（RL 电路）

代入三要素公式写出响应表达式

## 三、例题分析：阶跃响应求解

### 例 9：RL 电路阶跃响应

**已知**：$R_1 = R_2 = 8\Omega$，$R_3 = 6\Omega$，$L = 1\text{H}$，输入为单位阶跃电压 $\varepsilon(t)$

**求**：$i_2(t)$ 和 $u_L(t)$

#### 步骤 1：求初始值 $i_2(0_+)$、$u_L(0_+)$

零状态响应 $\Rightarrow i_2(0_+) = i_L(0_+) = 0$

$t = 0_+$ 时电感开路，电路为 $R_1$、$R_2$ 与 1V 电压源串联：

$$
u_L(0_+) = u_{R2} = \frac{R_2}{R_1 + R_2} \times 1 = \frac{8}{16} = 0.5\text{V}
$$

:::derivation
**初始值 $u_L(0_+)$ 的推导（分压法）**：

零状态条件下 $i_L(0_+) = i_L(0_-) = 0$（电感电流连续）。$t = 0_+$ 时电感相当于**开路**（电流为零），电路退化为 $R_1$、$R_2$ 与 1V 电压源串联。

由串联分压关系，$R_2$ 上分得的电压即电感两端电压：

$$u_L(0_+) = u_{R2} = \frac{R_2}{R_1 + R_2} \times U_S = \frac{8}{8+8} \times 1 = 0.5\text{V}$$

该方法适用于零状态下求解电感电压或电容电流的初始值：先将动态元件等效为开路（电感）或短路（电容），再用电阻电路方法求解。
:::

#### 步骤 2：求稳态值 $i_2(\infty)$、$u_L(\infty)$

$t \to \infty$ 时电感短路，设上方节点电压为 $u$，节点电压方程：

$$
\left( \frac{1}{R_1} + \frac{1}{R_2} + \frac{1}{R_3} \right) u = \frac{1}{R_1}
\Rightarrow \frac{20}{48}u = \frac{1}{8}
\Rightarrow u = \frac{3}{10}\text{V}
$$

:::derivation
**稳态值节点电压方程的推导（KCL）**：

$t \to \infty$ 时电感等效为**短路**（$u_L(\infty) = 0$，直流稳态下 $\mathrm{d}i/\mathrm{d}t = 0$）。设上方节点电压为 $u$，对节点列 KCL（流出为正）：

$$\frac{u - U_S}{R_1} + \frac{u}{R_2} + \frac{u}{R_3} = 0$$

整理得：

$$\left(\frac{1}{R_1} + \frac{1}{R_2} + \frac{1}{R_3}\right)u = \frac{U_S}{R_1}$$

代入 $R_1 = R_2 = 8\Omega$，$R_3 = 6\Omega$，$U_S = 1\text{V}$：

$$\left(\frac{1}{8} + \frac{1}{8} + \frac{1}{6}\right)u = \frac{1}{8} \Rightarrow \frac{20}{48}u = \frac{1}{8} \Rightarrow u = \frac{3}{10}\text{V}$$

该方法的关键是：直流稳态下电容开路、电感短路，将动态电路转化为纯电阻电路后用节点电压法求解。
:::

$$
i_2(\infty) = \frac{u}{R_3} = \frac{0.3}{6} = 0.05\text{A}, \quad u_L(\infty) = 0
$$

#### 步骤 3：求时间常数 $\tau$

移除电感，电压源置零，等效电阻：

$$
R_{eq} = R_1 \parallel R_2 + R_3 = \frac{8 \times 8}{8+8} + 6 = 10\Omega
$$

:::derivation
**等效电阻 $R_{eq}$ 的推导**：

求时间常数时需先求从动态元件两端看入的等效电阻。步骤：

1. 移除电感 $L$，得到有源一端口电路；
2. 独立源置零（电压源短路为 0V），此时 $R_1$ 与 $R_2$ 上端短接；
3. 从电感端口看入，$R_1$ 与 $R_2$ 并联后再与 $R_3$ 串联：

$$R_{eq} = (R_1 \parallel R_2) + R_3 = \frac{R_1 \cdot R_2}{R_1 + R_2} + R_3 = \frac{8 \times 8}{8+8} + 6 = 4 + 6 = 10\Omega$$

时间常数为 $\tau = L / R_{eq}$（RL 电路）或 $\tau = R_{eq} \cdot C$（RC 电路）。$R_{eq}$ 越大，RL 电路响应越快（$\tau$ 越小），RC 电路响应越慢（$\tau$ 越大）。
:::

$$
\tau = \frac{L}{R_{eq}} = \frac{1}{10} = 0.1\text{s}
$$

#### 步骤 4：写出响应表达式

$$
i_2(t) = \left[ 0.05 + (0 - 0.05)e^{-10t} \right] \varepsilon(t) = 0.05\left(1 - e^{-10t}\right) \varepsilon(t) \quad \text{A}
$$

$$
u_L(t) = \left[ 0 + (0.5 - 0)e^{-10t} \right] \varepsilon(t) = 0.5e^{-10t} \varepsilon(t) \quad \text{V}
$$

:::derivation
**零状态响应 $u_L(t)$ 的推导与分析**：

代入三要素 $u_L(0_+) = 0.5\text{V}$、$u_L(\infty) = 0\text{V}$（稳态时电感短路）、$\tau = 0.1\text{s}$：

$$u_L(t) = 0 + (0.5 - 0)e^{-10t} = 0.5e^{-10t} \text{ V}$$

该结果体现了电感电压的物理特性：

- $t = 0_+$ 时电感电流为零（开路），电压最大（$0.5\text{V}$，全部降在电感上）；
- 随时间推移，电感电流逐渐增大，电压按 $e^{-t/\tau}$ 指数衰减；
- $t \to \infty$ 时电感等效为短路，电压降为零。

注意：零状态响应中**电感电压的初始值不为零**（虽然电感电流初始值为零），这是因为电压取决于电流的变化率而非电流本身。
:::

## 四、全响应

### 4.1 基本概念

**全响应**：在非零原始状态下（$u_C(0_-) \neq 0$ 或 $i_L(0_-) \neq 0$），由储能元件的原始储能和独立源共同引起的响应。

### 4.2 全响应的分解（叠加定理）

$$
\boxed{\text{全响应} = \text{零输入响应} + \text{零状态响应}}
$$

:::derivation
**全响应 = 零输入响应 + 零状态响应 的推导（叠加定理）**：

设电路中含独立源和初始储能（$u_C(0_+) \neq 0$ 或 $i_L(0_+) \neq 0$）。利用**替代定理**，将电容用电压源 $u_C(0_+)$ 替代、电感用电流源 $i_L(0_+)$ 替代（替代后电路等效，端口特性不变）。

此时电路中存在两类电源：原独立源和替代电源（代表初始储能）。由**叠加定理**（线性电路中多个激励共同作用的响应 = 各激励单独作用响应之和）：

- **替代电源单独作用**（原独立源置零）：此时电路无外部激励，仅由初始储能驱动 → **零输入响应** $r_{zi}(t)$
- **原独立源单独作用**（替代电源置零，即初始储能为零）：此时电路为零状态 → **零状态响应** $r_{zs}(t)$

由叠加定理：

$$r(t) = r_{zi}(t) + r_{zs}(t)$$

即**全响应 = 零输入响应 + 零状态响应**。该分解体现了线性电路的根本特性，是三要素法适用于全响应的理论基础。
:::

**推导思路**：
1. 用电压源 $u_C(0_+)$ 替代电容，电流源 $i_L(0_+)$ 替代电感（替代电源）
2. 应用叠加定理：
   - 替代电源作用 + 原独立源置零 $\rightarrow$ 零输入响应
   - 原独立源作用 + 替代电源置零 $\rightarrow$ 零状态响应

### 4.3 全响应的三要素公式

一阶电路全响应同样满足：

$$
r(t) = r(\infty) + \left[ r(0_+) - r(\infty) \right] e^{-\frac{t}{\tau}}, \quad t \geq 0
$$

:::derivation
**全响应三要素公式的推导（由叠加分解推导）**：

由全响应分解：$r(t) = r_{zi}(t) + r_{zs}(t)$

- **零输入响应**（独立源为零，$r(\infty) = 0$，仅由初始值驱动）：

$$r_{zi}(t) = r(0_+)\,e^{-t/\tau}$$

- **零状态响应**（初始储能为零，$r(0_+) = 0$，仅由独立源驱动）：

$$r_{zs}(t) = r(\infty)\left(1 - e^{-t/\tau}\right)$$

叠加得全响应：

$$r(t) = r(0_+)\,e^{-t/\tau} + r(\infty)\left(1 - e^{-t/\tau}\right) = r(\infty) + \left[r(0_+) - r(\infty)\right]e^{-t/\tau}$$

可见全响应与零状态响应具有**相同的三要素公式形式**，区别仅在于：零状态响应中 $r(0_+) = 0$，而全响应中 $r(0_+) \neq 0$（需先求原始值再由换路定则确定初始值）。
:::

**与零状态响应的区别**：
- 零状态响应：$r(0_+) = 0$（动态元件初始储能为 0）
- 全响应：$r(0_+) \neq 0$（需先求原始值，再通过换路定则得初始值）

### 4.4 特殊情况验证

| 响应类型 | 条件 | 公式简化 |
|---------|------|---------|
| 零输入响应 | 独立源 = 0 | $r(t) = r(0_+) e^{-t/\tau}$（$r(\infty)=0$） |
| 零状态响应 | 原始储能 = 0 | $r(t) = r(\infty)(1 - e^{-t/\tau})$（$r(0_+)=0$） |

## 五、例题分析：全响应求解

### 例 13：RC 电路全响应

**已知**：开关闭合前电路已达稳态，$t=0$ 时开关闭合，$u_S = 25\text{V}$，$R_1 = 20\Omega$，$R_2 = 5\Omega$，$C$ 未知（$\tau = 4\text{s}$）

**求**：$u_C(t)$、$i_2(t)$

#### 步骤 1：求初始值

开关闭合前稳态，电容开路：
$$
u_C(0_-) = u_S = 25\text{V} \Rightarrow u_C(0_+) = 25\text{V}
$$

$t=0_+$ 时电容用 25V 电压源替代，对最外圈回路 KVL：
$$
5i_2(0_+) = u_C(0_+) \Rightarrow i_2(0_+) = \frac{25}{5} = 5\text{A}
$$

#### 步骤 2：求稳态值

$t \to \infty$ 时电容开路，电路为 $R_1$、$R_2$、$u_S$ 串联：
$$
20i_2(\infty) + 5i_2(\infty) - 25 = 0 \Rightarrow i_2(\infty) = 1\text{A}
$$
$$
u_C(\infty) = 5i_2(\infty) = 5\text{V}
$$

#### 步骤 3：时间常数

$$
R_{eq} = R_1 \parallel R_2 = \frac{20 \times 5}{20+5} = 4\Omega, \quad \tau = R_{eq}C = 4\text{s}
$$

#### 步骤 4：响应表达式

$$
u_C(t) = 5 + (25 - 5)e^{-t/4} = 5 + 20e^{-0.25t} \quad \text{V}, \quad t \geq 0
$$

:::derivation
**全响应 $u_C(t)$ 的三要素法推导**：

将三要素 $u_C(0_+) = 25\text{V}$、$u_C(\infty) = 5\text{V}$、$\tau = 4\text{s}$ 代入三要素公式 $r(t) = r(\infty) + [r(0_+) - r(\infty)]e^{-t/\tau}$：

$$u_C(t) = 5 + (25 - 5)e^{-t/4} = 5 + 20e^{-0.25t} \text{ V}$$

该结果可分解为：
- **稳态分量**（强制响应）：$u_C(\infty) = 5\text{V}$（电容开路后由分压决定）
- **暂态分量**（自由响应）：$20e^{-0.25t}\text{V}$（按指数衰减，衰减速率由 $\tau$ 决定）

$t = 0$ 时暂态分量最大（20V），随时间衰减至零，最终 $u_C$ 趋向稳态值 5V。也可从分解角度验证：零输入响应 $25e^{-t/4}$ + 零状态响应 $5(1 - e^{-t/4})$ = $5 + 20e^{-t/4}$ ✓
:::

$$
i_2(t) = 1 + (5 - 1)e^{-t/4} = 1 + 4e^{-0.25t} \quad \text{A}, \quad t \geq 0
$$

> 注：也可由 $i_2(t) = u_C(t)/R_2$ 直接验证结果一致性。

## 六、总结

### 6.1 三要素法核心要点

1. **初始值**：零状态响应中 $u_C(0_+)=0$、$i_L(0_+)=0$；全响应需先求 $u_C(0_-)$、$i_L(0_-)$
2. **稳态值**：电容开路、电感短路，转化为电阻电路求解
3. **时间常数**：$\tau = R_{eq}C$ 或 $\tau = L/R_{eq}$，$R_{eq}$ 为动态元件移除后、独立源置零的端口等效电阻

### 6.2 公式对比

| 响应类型 | 通用公式 | 适用条件 |
|---------|---------|---------|
| 零输入响应 | $r(t) = r(0_+) e^{-t/\tau}$ | 独立源 = 0 |
| 零状态响应 | $r(t) = r(\infty)(1 - e^{-t/\tau})\varepsilon(t)$ | 原始储能 = 0 |
| 全响应 | $r(t) = r(\infty) + [r(0_+) - r(\infty)]e^{-t/\tau}$ | 一般情况 |

### 6.3 解题建议

- 先判断响应类型（零输入/零状态/全响应）
- 严格按"初始值→稳态值→时间常数→代入公式"四步求解
- 复杂电路优先使用节点电压法或戴维宁等效简化计算
