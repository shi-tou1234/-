# 双极结型三极管（BJT）核心知识精讲

> 💡 **前言**：模拟电子技术的核心基石。初学时觉得公式多、概念抽象很正常。本篇抛开死记硬背，带你**从物理本质出发，一步步推导出电路模型和公式**。配合水流、水阀等生动类比，彻底吃透三极管！

# 第一课：BJT的“内脏”与工作原理 (对应书本 5.1)

## 1. 结构大揭秘：为什么叫“双极型”、“三极管”？
- **双极型 (Bipolar)**：因为管子内部既有**自由电子**，又有**空穴**，这两种极性的载流子同时参与导电。
- **结构**：想象一个三明治，两块面包夹着一块薄薄的火腿。
  - **NPN型**：两边是N型半导体（电子多），中间夹着P型半导体（空穴多）。
  - **PNP型**：两边是P型，中间夹着N型。（原理相同，仅电流方向与电压极性相反，本文以NPN为例）
- **三个区与三个极**：
  1. **发射区 (Emitter, e)**：掺杂浓度极高（杂质超级多），就像一个蓄水池，准备发射大量电子。
  2. **基区 (Base, b)**：极薄（微米级），且掺杂浓度很低。就像一座很窄的桥。
  3. **集电区 (Collector, c)**：面积很大，用来收集电子。
- **两个结**：这三个区交界处形成了两个PN结。e和b之间叫**发射结**，c和b之间叫**集电结**。

> 👨‍🏫 **老师敲黑板**：三极管绝对不是把两个二极管背靠背连在一起就能成的！它的核心在于**基区非常非常薄**，这决定了它能放大电流。

## 2. 放大状态下的微观世界（核心机理）
三极管有放大、饱和、截止三种状态。要让它成为一个放大器，必须满足两个偏置条件：
**发射结正偏，集电结反偏。**

想象一下 NPN 管内部发生了什么：
1. **发射 (注入)**：发射结正偏，发射区的海量电子像决堤一样涌入基区，形成发射极电流 $I_E$。
2. **复合与传输**：电子来到基区。因为基区是P型（有空穴），少部分电子会掉进空穴（复合），这部分复合掉的电子由基极电源抽走补充，形成微小的基极电流 $I_B$。**但由于基区太薄了，98%以上的电子根本来不及复合，就由于浓度差继续向集电结跑去**。
3. **收集**：集电结是反偏的，反偏的PN结内部电场方向刚好能把跑过来的电子瞬间“吸”过去！这些被收集的电子形成了集电极电流 $I_C$。

## 3. 电流分配法则与放大系数（公式推导）
基于上面的物理过程，我们得出最基础的电流守恒定律：
$$ I_E = I_B + I_C $$

:::derivation
**电流守恒定律推导**（$I_E = I_B + I_C$）：

在 NPN 三极管中，发射极电流 $I_E$ 是发射区注入基区的电子流（由发射结正偏驱动）。

这些电子进入基区后有两条去向：

1. **少量复合**：与基区的空穴复合，被基极电源抽走补充，形成基极电流 $I_B$（约占 $1\sim2\%$）。

2. **多数穿越**：由于基区极薄，绝大多数电子来不及复合就穿过基区到达集电结，被集电结反偏电场收集，形成集电极电流 $I_C$（约占 $98\sim99\%$）。

由电荷守恒定律，注入基区的总电子数等于复合数加收集数：

$$I_E = I_B + I_C$$

这是三极管最基本的电流关系，无论工作状态如何均成立。
:::

- **共基极电流放大系数 $\alpha$**：到达集电极的电流占总发射电流的比例。因为绝大部分电子都过来了，所以 $\alpha$ 通常在 0.98 以上。
  $$ I_C = \alpha I_E + I_{CBO} $$

  :::derivation
  **共基极电流放大系数 $\alpha$ 公式推导**：

  在放大状态下，发射区注入基区的电子绝大部分被集电结收集。定义 $\alpha$ 为到达集电极的电流占发射电流的比例：

  $$\alpha = \frac{I_C - I_{CBO}}{I_E}$$

  其中 $I_{CBO}$ 是集电结反偏时少数载流子漂移产生的漏电流（与发射极无关）。

  移项得：

  $$I_C = \alpha I_E + I_{CBO}$$

  由于基区极薄、掺杂浓度低，$98\%$ 以上的电子都能穿越基区被收集，故 $\alpha \approx 0.98\sim0.995$。忽略 $I_{CBO}$（通常为纳安级），近似为 $I_C \approx \alpha I_E$。
  :::

  *(注：$I_{CBO}$ 是集电结反偏产生的极微小的漏电流，通常忽略)*
- **共射极电流放大系数 $\beta$**：集电极电流与基极电流的比例。
  把 $I_E = I_B + I_C$ 代入上面公式：
  $I_C = \alpha (I_B + I_C) \Rightarrow I_C (1-\alpha) = \alpha I_B \Rightarrow I_C = \frac{\alpha}{1-\alpha} I_B$
  我们把 $\frac{\alpha}{1-\alpha}$ 定义为 $\beta$。假如 $\alpha = 0.99$，那么 $\beta = \frac{0.99}{0.01} = 99$。
  $$ I_C = \beta I_B + I_{CEO} $$

  :::derivation
  **共射极电流放大系数 $\beta$ 公式推导**：

  目标：将 $I_C$ 表示为 $I_B$ 的函数（共射极组态中 $I_B$ 是输入、$I_C$ 是输出）。

  **步骤 1**：由 $I_C = \alpha I_E + I_{CBO}$ 和电流守恒 $I_E = I_B + I_C$，代入消去 $I_E$：

  $$I_C = \alpha(I_B + I_C) + I_{CBO}$$

  **步骤 2**：展开并合并同类项：

  $$I_C = \alpha I_B + \alpha I_C + I_{CBO}$$

  $$I_C - \alpha I_C = \alpha I_B + I_{CBO}$$

  $$I_C(1 - \alpha) = \alpha I_B + I_{CBO}$$

  **步骤 3**：解出 $I_C$：

  $$I_C = \frac{\alpha}{1-\alpha} I_B + \frac{I_{CBO}}{1-\alpha}$$

  **步骤 4**：定义 $\beta = \dfrac{\alpha}{1-\alpha}$，并记 $I_{CEO} = \dfrac{I_{CBO}}{1-\alpha} = (1+\beta)I_{CBO}$（穿透电流），得：

  $$I_C = \beta I_B + I_{CEO}$$

  **数值验证**：$\alpha=0.99$ 时 $\beta=\dfrac{0.99}{0.01}=99$，说明微小基极电流变化能引起近百倍的集电极电流变化。
  :::

  *(注：$I_{CEO} = (1+\beta)I_{CBO}$ 称为穿透电流。)*

> 👨‍🏫 **老师总结**：这就是三极管放大的本质——**以小控大**。微小的基极电流 $I_B$（像水阀），控制着巨大的集电极电流 $I_C$（主水流）。只要改变一点点 $I_B$，$I_C$ 就会成几十上百倍地成比例变化。它是**电流控制器件**。

# 第二课：看懂三极管的“体检报告”——特性曲线 (对应 5.1.3)

我们要把三极管当作一个“黑盒”二端口网络来看。以最常用的**共射极组态**（发射极作为公共接地端，基极输入，集电极输出）为例。

## 1. 输入特性曲线 $i_B = f(v_{BE})|_{v_{CE}=常数}$
- **长什么样？** 跟二极管的正向导通曲线一模一样！因为输入端就是正偏的发射结。
- **开启电压**：硅管大约 $0.5V$ 开始导通，正常工作时 $V_{BE}$ 约为 $0.6V \sim 0.7V$。

## 2. 输出特性曲线 $i_C = f(v_{CE})|_{i_B=常数}$ （🌟极其重要！）
这张图分为三个区，决定了三极管的命运：
1. **放大区**（曲线平坦的部分）：
   - **条件**：发射结正偏（$v_{BE} > 0.5V$），集电结反偏（$v_{CE} \ge v_{BE}$）。
   - **特点**：此时 $i_C$ 几乎只受 $i_B$ 控制，跟 $v_{CE}$ 没多大关系。体现了恒流源特性（$i_C = \beta i_B$）。
   - *(注：为什么曲线会有微微上翘？这叫**基区宽度调制效应 / 厄尔利效应**。$v_{CE}$ 越大，集电结越宽，挤压基区使其更薄，复合减少，$\beta$ 变大一点点。反向延长交于一点即 Early 电压 $V_A$)*。
2. **饱和区**（最左边陡峭的部分）：
   - **条件**：发射结正偏，集电结也正偏了！（$v_{CE} < v_{BE}$，通常 $v_{CE} < 0.3V$）。
   - **特点**：此时集电极不仅不收集电子，反而向基区发射电子。$i_C$ 不再受 $i_B$ 控制，无论 $i_B$ 怎么变，$i_C$ 都上不去了。相当于水龙头开到最大，水管水压不够了。管压降叫 $V_{CES}$（约 0.3V）。
3. **截止区**（最下面的 $i_B = 0$ 曲线及以下）：
   - **条件**：发射结反偏或两端电压小于开启电压。
   - **特点**：管子关断，$i_C \approx 0$。

# 第三课：如何让三极管干活？——基本共射放大电路 (对应 5.2 - 5.3)

## 1. 电路长啥样？为什么要加直流电源？
想象一下，交流信号（比如麦克风传来的声音）是有正有负的正弦波。但三极管输入端像二极管，负半周它就截止（罢工）了！
**解决办法**：用直流电源（$V_{CC}, V_{BB}$）给三极管提供一个直流的“底子”，把整个交流信号“垫高”，让管子始终处于正偏导通的放大区。

这就引出了放大电路分析的核心思想：**交直流分开分析！**
- **直流通路**：只看直流，用来计算**静态工作点（Q点）**。电容视为开路。
- **交流通路**：只看交流，用来计算**放大倍数、输入输出电阻**。电容视为短路，直流电源（$V_{CC}$）视为短路（交流地）。

## 2. 静态分析（找 Q 点）
Q点包含四个量：$I_{BQ}, I_{CQ}, V_{BEQ}, V_{CEQ}$。
用**估算法**（这是工程师最常用的手法）：
1. 看输入回路求基极电流：$I_{BQ} = \frac{V_{BB} - V_{BEQ}}{R_b}$ （一般默认 $V_{BEQ} = 0.7V$）
2. 看集电极电流：$I_{CQ} = \beta I_{BQ}$
3. 看输出回路求管压降：$V_{CEQ} = V_{CC} - I_{CQ}R_c$

> 👨‍🏫 **图解法的意义与失真**：
> 如果 Q 点设得太高（$I_B$ 太大），交流信号一来，管子容易进入**饱和区**，正半周波形会被削平，叫**饱和失真**。
> 如果 Q 点设得太低，交流信号负半周时管子就**截止**了，底部波形被削平，叫**截止失真**。
> 最佳 Q 点应该在交流负载线的中点。

## 3. 动态分析（微变等效电路法 / 小信号模型）🌟最难懂的公式在这里！
当交流信号很小的时候，我们可以把特性曲线上的一小段看作直线。这时，可以用一个线性电路来代替三极管！这就是**H参数小信号模型**。

- **输入端相当于什么？** 一个电阻！叫 **$r_{be}$**。
  - 推导：基极输入，发射极接地。电流从 b 到 e 经过了什么？首先是基区自身的电阻 $r_{bb'}$（约 200欧）；然后经过发射结，发射结类似二极管，它的动态电阻 $r_e = \frac{26mV}{I_{EQ}}$；但注意，流过 $r_e$ 的电流是 $I_E$，而我们从基极看进去的电流是 $I_B$。因为 $I_E = (1+\beta)I_B$，所以在基极看来，这个结电阻放大了 $(1+\beta)$ 倍。
  - **核心公式**： 
    $$r_{be} = r_{bb'} + (1+\beta)\frac{26mV}{I_{EQ} (mA)}$$

    :::derivation
    **输入电阻 $r_{be}$ 公式推导**：

    基极到发射极的等效电阻由两部分串联组成。

    **第一部分：基区体电阻 $r_{bb'}$**

    这是基区半导体材料自身的电阻，由基区掺杂浓度和几何尺寸决定，硅管典型值约 200$\Omega$。

    **第二部分：发射结等效电阻（折算到基极）**

    发射结正偏时等效于一个二极管，其动态电阻由 PN 结伏安特性 $I=I_S(e^{V/V_T}-1)$ 求导得到：

    $$r_e = \frac{dV}{dI} = \frac{V_T}{I_{EQ}} = \frac{26\text{mV}}{I_{EQ}\text{(mA)}}$$

    其中 $V_T = \dfrac{kT}{q}\approx 26\text{mV}$（室温下的热电压）。

    **关键折算**：$r_e$ 上流过的电流是 $I_E$，但从基极看进去的电流是 $I_B$。由 $I_E = (1+\beta)I_B$，若基极电流变化 $\Delta I_B$，则发射极电流变化 $\Delta I_E = (1+\beta)\Delta I_B$。

    在 $r_e$ 上的压降为 $\Delta V = r_e \cdot \Delta I_E = r_e(1+\beta)\Delta I_B$。

    从基极看进去的等效电阻为：

    $$r_{e,\text{折算}} = \frac{\Delta V}{\Delta I_B} = (1+\beta) r_e = (1+\beta)\frac{26\text{mV}}{I_{EQ}}$$

    **总电阻**：两部分串联：

    $$r_{be} = r_{bb'} + (1+\beta)\frac{26\text{mV}}{I_{EQ}\text{(mA)}}$$

    **物理意义**：发射结电阻被"反射"到基极后放大了 $(1+\beta)$ 倍，这是因为基极电流远小于发射极电流。
    :::
- **输出端相当于什么？** 一个受控电流源！大小为 **$\beta i_b$**。方向向下（流向发射极）。

**开始算交流指标（背熟这套流程）：**
将原图中的三极管替换为小信号模型（b、e之间接 $r_{be}$，c、e之间接向下的 $\beta i_b$），画出交流等效图。
1. **电压增益 $A_v$**：
   - 输出电压 $v_o$ 产生在负载上（$R_c \parallel R_L$，记为 $R_L'$）。谁在流过它？是集电极电流 $-i_c = -\beta i_b$ （负号表示电流从地往上流，使得 $v_o$ 为负极性）。
   - 所以：$v_o = -\beta i_b (R_c \parallel R_L)$
   - 输入电压 $v_i$ 加在输入端：$v_i = i_b r_{be}$
   - 相除得增益：
     $$A_v = \frac{v_o}{v_i} = -\frac{\beta (R_c \parallel R_L)}{r_{be}}$$

     :::derivation
     **共射极电压增益公式推导**：

     基于小信号等效电路（b-e 间接 $r_{be}$，c-e 间接受控电流源 $\beta i_b$）。

     **步骤 1：求输出电压 $v_o$**

     集电极交流电流 $i_c = \beta i_b$，方向从 c 流向 e（向下）。负载 $R_L$ 与 $R_c$ 并联（交流通路中 $V_{CC}$ 接地），记 $R_L' = R_c \parallel R_L$。

     输出电压 $v_o$ 是集电极对地的电压。电流 $\beta i_b$ 从地向上流过 $R_L'$（因为受控源方向向下，按 KCL 负载电流方向相反），故：

     $$v_o = -\beta i_b \cdot R_L' = -\beta i_b (R_c \parallel R_L)$$

     负号表示输出与输入反相。

     **步骤 2：求输入电压 $v_i$**

     输入电压加在 $r_{be}$ 上：

     $$v_i = i_b \cdot r_{be}$$

     **步骤 3：求增益**

     $$A_v = \frac{v_o}{v_i} = \frac{-\beta i_b (R_c \parallel R_L)}{i_b \cdot r_{be}} = -\frac{\beta (R_c \parallel R_L)}{r_{be}}$$

     **物理意义**：
     - $\beta$ 越大、负载越大 $\rightarrow$ 增益越高；
     - $r_{be}$ 越大 $\rightarrow$ 增益越低（信号源内阻分压）；
     - 负号 $\rightarrow$ 共射电路是反相放大器。
     :::
     *(负号代表反相！你大喊一声正弦波，喇叭里出来的是倒过来的正弦波，但幅度被放大了。)*
2. **输入电阻 $R_i$**：
   从输入端看进去，有基极偏置电阻 $R_b$ 和三极管的 $r_{be}$ 并联。
   $$R_i = R_b \parallel r_{be}$$

   :::derivation
   **输入电阻公式推导**（$R_i = R_b \parallel r_{be}$）：

   输入电阻定义为从放大电路输入端看进去的等效电阻：$R_i = \dfrac{v_i}{i_i}$。

   在共射放大电路的交流通路中，输入端到地之间有两条并联路径：

   1. **偏置电阻 $R_b$**：直接连接在基极与地（交流地）之间。
   2. **三极管输入电阻 $r_{be}$**：即基极到发射极的等效电阻，发射极接地。

   两条路径并联，故输入电阻为：

   $$R_i = R_b \parallel r_{be} = \frac{R_b \cdot r_{be}}{R_b + r_{be}}$$

   由于通常 $R_b \gg r_{be}$（偏置电阻远大于三极管输入电阻），故 $R_i \approx r_{be}$。

   **设计启示**：$R_b$ 过大会使输入电阻主要由 $r_{be}$ 决定（通常仅 1k$\Omega$ 左右），这限制了放大器对高内阻信号源的适配能力。
   :::
3. **输出电阻 $R_o$**：
   令输入信号源为0，此时 $i_b = 0$，受控源断开。从输出端看进去，只剩集电极电阻。
   $$R_o \approx R_c$$

   :::derivation
   **输出电阻公式推导**（$R_o \approx R_c$）：

   输出电阻的定义：令输入信号源 $v_s = 0$（短路），在输出端施加测试电压 $v_t$，测量流入电流 $i_t$，则 $R_o = \dfrac{v_t}{i_t}$。

   **步骤 1**：令 $v_s = 0$，则基极交流电流 $i_b = 0$。

   **步骤 2**：受控电流源 $\beta i_b = \beta \cdot 0 = 0$，相当于受控源**断路**（开路）。

   **步骤 3**：从输出端看进去，集电极到地之间只剩 $R_c$（受控源开路后，三极管 c-e 间无通路）。

   故：

   $$R_o = \frac{v_t}{i_t} = R_c$$

   **近似说明**：严格来说，三极管输出端还有 $r_{ce}$（集电极-发射极间动态电阻，由基区宽度调制效应引起），与 $R_c$ 并联。但 $r_{ce}$ 通常很大（数十千欧以上），远大于 $R_c$（数千欧），故 $R_o = r_{ce} \parallel R_c \approx R_c$。
   :::

# 第四课：驯服温度恶魔——静态工作点稳定电路 (对应 5.4)

## 1. 为什么 Q 点会跑？
温度是半导体的死敌。温度一升高：
- 穿透电流 $I_{CEO}$ 翻倍增加。
- 发射结压降 $V_{BE}$ 降低（每升高1℃降 2mV）。
- $\beta$ 值变大。
这三点都会导致集电极静态电流 $I_{CQ}$ 疯狂飙升。Q点一升高，直接进入饱和失真！

## 2. 拯救者：分压式射极偏置电路（经典神级电路）
书上的图 5.4.1a。这个电路的设计极其巧妙，它引入了**直流负反馈**。
- **改造1：分压定基极电位**。用 $R_{b1}$ 和 $R_{b2}$ 给基极分压。设计时让流过它们的电流远大于基极电流 $I_B$。这样，基极电位 $V_{BQ} \approx \frac{R_{b2}}{R_{b1}+R_{b2}}V_{CC}$，死死钉住，不受管子内部温度变化影响。
- **改造2：发射极接电阻 $R_e$**。这是灵魂所在！

**稳定过程（魔法时刻）：**
温度升高 $\rightarrow$ $I_{CQ}$ 企图增加 $\rightarrow$ $I_{EQ}$ 跟着增加 $\rightarrow$ 发射极电位 $V_{EQ} = I_{EQ}R_e$ 升高！
但是，基极电位 $V_{BQ}$ 被电阻分压钉死了是不变的。
那么，加在发射结上的电压 $V_{BE} = V_{BQ} - V_{EQ}$ 就会**被动减小**！
$V_{BE}$ 减小 $\rightarrow$ 基极电流 $I_B$ 减小 $\rightarrow$ 强行把 $I_{CQ}$ 拉回原位。
*(这叫负反馈：你想变大，我就产生一个机制压制你变大。$R_e$ 越大，反馈越强，越稳定)*

## 3. 付出的代价与弥补
- **代价**：交流信号来了也会经历这个过程，导致交流放大倍数剧降！
  加上 $R_e$ 后的增益公式会变成：$A_v \approx -\frac{\beta R_L'}{r_{be} + (1+\beta)R_e}$，因为分母多了一大块，增益极小。
- **弥补**：在 $R_e$ 旁边并联一个**大电容 $C_e$（旁路电容）**。
  直流时，电容开路，$R_e$ 稳定 Q 点；交流时，大电容相当于短路，把 $R_e$ 短接了，交流信号全从电容走，不触发负反馈。这样增益又恢复到了：
  $$A_v = -\frac{\beta (R_c \parallel R_L)}{r_{be}}$$
  完美！

  :::derivation
  **带旁路电容的增益恢复推导**：

  **无旁路电容时**（$R_e$ 存在于交流通路），发射极电阻引入交流负反馈：

  1. 输入电压：$v_i = i_b r_{be} + (1+\beta)i_b R_e = i_b[r_{be} + (1+\beta)R_e]$

     （发射极电流 $i_e = (1+\beta)i_b$ 流过 $R_e$ 产生反馈压降）

  2. 输出电压：$v_o = -\beta i_b (R_c \parallel R_L)$

  3. 增益：

  $$A_v = \frac{-\beta i_b (R_c \parallel R_L)}{i_b[r_{be}+(1+\beta)R_e]} = -\frac{\beta(R_c\parallel R_L)}{r_{be}+(1+\beta)R_e}$$

  分母多了 $(1+\beta)R_e$（发射极电阻被放大 $(1+\beta)$ 倍），增益大幅下降。

  **加旁路电容 $C_e$ 后**：$C_e$ 对交流信号相当于短路（大电容容抗 $X_C = \dfrac{1}{\omega C}\approx 0$），将 $R_e$ 短接。交流等效电路中 $R_e$ 消失，增益恢复为：

  $$A_v = -\frac{\beta(R_c\parallel R_L)}{r_{be}}$$

  **设计思想**：$C_e$ 实现"直流负反馈稳定 Q 点、交流无负反馈保增益"的双重目标，是模拟电路中"隔直通交"思想的典型应用。
  :::

# 第五课：三极管的其他姿势——共集和共基电路 (对应 5.5)

除了发射极接地（共射），我们还能怎么接？

## 1. 共集电极放大电路 (射极跟随器)
- **接法**：集电极接电源（交流地），基极输入，**发射极输出**。负载 $R_L$ 接在发射极。
- **特点分析**（想象你拉着一个弹簧箱子）：
  - **电压增益 $A_v \approx 1$ 且同相**。发射极电压死死咬住基极电压（差个固定的 0.7V），输入涨1V，输出也涨1V，所以叫**电压跟随器**。
  - **输入电阻极高**。为什么？你看进去不仅看到了 $r_{be}$，还看到了被放大了 $(1+\beta)$ 倍的发射极负载 $R_L$。公式：
    $$R_i = R_b \parallel [r_{be} + (1+\beta)(R_e \parallel R_L)]$$

    :::derivation
    **共集电极电路输入电阻公式推导**：

    共集电极电路（射极跟随器）的输入电阻远高于共射电路，原因在于发射极负载被"反射"到基极后放大了 $(1+\beta)$ 倍。

    **推导步骤**：

    1. **从基极看进去的等效电阻**：基极电流 $i_b$ 流经 $r_{be}$，同时发射极电流 $i_e = (1+\beta)i_b$ 流过发射极等效负载 $R_e' = R_e \parallel R_L$。

    2. **输入电压**：

    $$v_i = v_{be} + v_e = i_b \cdot r_{be} + i_e \cdot R_e'$$

    $$= i_b \cdot r_{be} + (1+\beta)i_b \cdot R_e' = i_b[r_{be} + (1+\beta)R_e']$$

    3. **三极管输入电阻**：

    $$r_{in} = \frac{v_i}{i_b} = r_{be} + (1+\beta)(R_e \parallel R_L)$$

    4. **总输入电阻**（再与偏置电阻 $R_b$ 并联）：

    $$R_i = R_b \parallel [r_{be} + (1+\beta)(R_e \parallel R_L)]$$

    **关键洞察**：发射极负载 $R_e'$ 流过的是 $i_e = (1+\beta)i_b$，从基极看进去等效于放大了 $(1+\beta)$ 倍。这正是"折算"思想——将发射极回路的电阻反射到基极回路时乘以 $(1+\beta)$。

    **比喻**：就像 100 个人帮你扛重物，你只需出 1/100 的力，感觉物体变轻了（电阻变大），故输入电阻极高。
    :::
    这就像你推箱子，有 100 个人（$\beta$）在帮你扛重物，你感觉箱子特别轻（电阻大，抽取的输入电流极小）。
  - **输出电阻极低**。带负载能力极强。
- **用途**：不能放大电压，用来干嘛？作多级放大器的**输入级**（减轻信号源负担），**输出级**（带得动大喇叭等重负载），或者**缓冲级**。

## 2. 共基极放大电路
- **接法**：基极接地，**发射极输入**，集电极输出。
- **特点**：
  - **没有电流放大**（$I_C \approx I_E$，增益 $\alpha \approx 1$）。
  - **有电压放大**，且同相。公式与共射极类似：$A_v = \frac{\beta R_L'}{r_{be}}$。
  - **输入电阻极小**（几十欧姆）。
- **用途**：这玩意儿高频特性超级好（因为避开了极间电容的密勒效应，第六章会学），专用于高频宽带放大。

# 第六课：强强联手——多级放大与复合管 (对应 5.7)

单兵作战能力有限，我们要把多个放大器串起来。

## 1. 多级放大的核心原则
- **总电压增益**等于各级增益的乘积：$A_v = A_{v1} \cdot A_{v2} \dots$
- **⚠️最容易犯的错误**：算前一级的增益时，**必须把后一级的输入电阻作为前一级的负载！** （即 $R_{L1}' = R_{c1} \parallel R_{i2}$）。前后级是紧密相连互相影响的。
- **输入/输出电阻**：整个系统的输入电阻就是第一级的输入电阻；输出电阻就是最后一级的输出电阻。

## 2. 复合管（达林顿管）
有时候我们需要极大的 $\beta$ 值（比如驱动大电机），买不到怎么办？
用两个同类型的管子（比如两个 NPN），前一个管子的发射极接到后一个管子的基极。
- **魔法效果**：总放大倍数 $\beta \approx \beta_1 \cdot \beta_2$。如果一个是 100，另一个是 100，组合起来就是 10000！
- **副作用**：穿透电流也放大了，温度稳定性变差，且开启电压变成了两个管子之和（约 1.4V）。

# 总结篇：FET（场效应管）与 BJT 的巅峰对决 (对应 5.6)

学完 BJT，你肯定想起了之前学的 MOS 管。它们俩谁更牛？
- **控制原理**：MOS 是 **电压控制**（栅极不耗电，极高输入阻抗）；BJT 是 **电流控制**（基极要吃掉一部分电流）。
- **敏感度**：BJT 的电流是随电压**指数**变化的，MOS 只是**平方**关系。所以 BJT 跨导更大，对微小信号放大能力更强。
- **使用场景**：MOS 管功耗低、容易做到纳米级，现在所有的电脑 CPU、手机芯片全是 MOS 管组成的集成电路。而 BJT 驱动能力强、速度快，多用于分立元件的大功率放大、模拟高频电路中。
