## 1. 齐次定理与叠加定理

### 1.1 线性电路与线性性质

在介绍定理之前，首先需要明确**线性电路**的概念：由独立电源和线性元件（线性电阻、线性受控源等）组成的电路称为线性电路。线性电路具有两个基本性质：**齐次性**与**可加性**，两者合称为**叠加性**。

设激励 $x$（可以是电压源电压或电流源电流）与响应 $y$（可以是电路中任意支路的电压或电流）之间的函数关系为 $y = f(x)$，则：

- **齐次性**：若激励增大或减小 $k$ 倍，则响应也同比例变化，即 $f(kx) = k \cdot f(x)$。
- **可加性**：多个激励共同作用产生的响应，等于各激励单独作用时所产生的响应之和，即 $f(x_1 + x_2) = f(x_1) + f(x_2)$。

两者统一起来，即：
$$
f(ax_1 + bx_2) = a \cdot f(x_1) + b \cdot f(x_2)
$$
其中 $a$、$b$ 为任意常数。
### 1.2 叠加定理的内容与证明

#### 定理表述

**叠加定理**：在线性电路中，任一支路的电流或电压，等于电路中每一个独立电源单独作用于电路时，在该支路所产生的电流或电压的代数和。

这里的“单独作用”是指：当考虑某一个独立电源作用时，其他独立电源均视为“不作用”——电压源短路、电流源开路。

#### 定理的证明思路

叠加定理的成立，根源于电路基本定律（基尔霍夫定律）的线性本质。对于任意一个具有 $b$ 条支路的线性电阻电路，其支路电流可由节点电压方程或回路电流方程来描述，这些方程都是一组线性代数方程：

$$
\mathbf{A} \cdot \mathbf{X} = \mathbf{B}
$$

其中，系数矩阵 $\mathbf{A}$ 仅由电路中的电阻和受控源参数决定，与独立源无关；列向量 $\mathbf{B}$ 是各独立源贡献的线性组合。根据线性代数理论，方程的解 $\mathbf{X}$（即各支路的电压或电流）等于每个独立源单独贡献的线性叠加。这正是叠加定理的数学本质。

### 1.3 齐次定理

**齐次定理**（也称齐性定理）：在只有一个激励作用的线性电路中，响应与激励成正比。当激励增大或减小 $n$ 倍时，任意一处的电压和电流响应也同时增大或减小 $n$ 倍。

- **公式表达**：若 $X$ 为响应，$J$ 为激励，则 $X = kJ$（$k$ 为常数，由电路结构决定）。
- **多激励情况**：当**所有**独立源同时增大 $n$ 倍时，响应也增大 $n$ 倍。但注意，若只有部分独立源变化，则需用叠加定理分别计算。

齐次定理可视为叠加定理在单激励情形下的特例，也是线性电路定义中“齐次性”的直接体现。它在分析T型电路、梯形网络等具有单一激励的电路时尤为便捷。

### 1.4 应用方法、注意事项与典型例题

#### 应用步骤
1. **分解电路**：让每个独立源逐个作用，画出各分电路图。
2. **置零处理**：不作用的电压源 → 短路（导线）；不作用的电流源 → 开路（断开）。
3. **受控源处理**：**受控源必须保留**在各分电路中，不能置零。其控制变量随独立源的变化而变化。
4. **叠加求和**：将各分电路中的响应分量进行代数叠加。注意参考方向——与原参考方向一致取正号，相反取负号。

#### 重要注意事项

| 序号 | 注意点 | 说明 |
|:---:|:---|:---|
| ① | **只适用于线性电路** | 非线性电路（含二极管、晶体管等非线性元件）不适用。 |
| ② | **功率不能直接叠加** | 功率 $P = UI$ 或 $P = I^2R$，与电压/电流是平方关系，不是线性关系。应先叠加求出总电压或总电流，再计算功率。|
| ③ | **受控源始终保留** | 在各分电路中，受控源必须保留，不能像独立源那样置零。 |
| ④ | **注意参考方向** | 叠加时按参考方向取代数和的“+”“-”号，方向一致为正，相反为负。 |
| ⑤ | **独立源分组作用** | 当电路中有多个同类型电源时，可以将它们分组作用以提高效率。 |

#### 典型例题解析

**例题**：试用叠加定理计算下图中3Ω电阻支路的电流 $I$。

电路结构：电路中包含一个4V电压源、一个6V电压源，以及2Ω、6Ω、3Ω三个电阻，其中3Ω电阻为待求支路，两个电压源分别位于不同回路中。

**解题思路**：
1. **分电路一**：4V电压源单独作用。将6V电压源短路，画出等效电路。此时6Ω电阻与3Ω电阻并联后与2Ω电阻串联，通过分压分流关系求出流经3Ω电阻的电流分量 $I_1$。
2. **分电路二**：6V电压源单独作用。将4V电压源短路，画出等效电路。此时2Ω电阻与3Ω电阻并联后与6Ω电阻串联，求出流经3Ω电阻的电流分量 $I_2$。
3. **叠加**：总电流 $I = I_1 + I_2$（注意参考方向，若 $I_2$ 的实际方向与参考方向相反，则取负号）。


## 2. 替代定理

### 2.1 定理内容

**替代定理**：给定任意一个具有唯一解的电路，若已知其中某条支路的支路电压 $u_k$ 和支路电流 $i_k$，则该支路可以用一个电压值 $u_S = u_k$ 的**独立电压源**替代，或者用一个电流值 $i_S = i_k$ 的**独立电流源**替代。替代后，电路中其余部分的全部支路电压和支路电流均保持原值不变。

替代定理的适用范围很广——它不要求电路必须是线性的，对非线性电路、时变电路也同样适用，前提是电路存在唯一解。

### 2.2 适用条件与注意事项

| 条件 | 说明 |
|:---|:---|
| **电路具有唯一解** | 替代定理成立的前提是原电路有唯一解，否则替代后可能改变解的性质。 |
| **无耦合关系** | 被替代的一端口子电路与原电路的剩余部分**无耦合关系**。例如，若被替代支路中含有受控源，而其控制量在剩余电路中，则替代后控制关系会丢失，定理不适用。 |
| **参考方向一致** | 替代电压源的极性与 $u_k$ 一致，替代电流源的参考方向与 $i_k$ 一致。 |
| **替代前后解的唯一性** | 替代后电路仍应具有唯一解。若替代导致电路出现多解或无解（如将电压源并联的电阻替换为电压源而改变电路拓扑），则替代无效。 |

### 2.3 解题技巧与典型应用

- **已知端口电压** → 用**电压源**替代，简化后续计算。
- **已知端口电流** → 用**电流源**替代。
- **电压电流均已知** → 两者皆可，根据计算方便选择。

替代定理的真正价值在于：当电路中某一部分的电压或电流已经通过某种方式求出后，可用电源将其替换，使电路结构简化，再配合回路分析法或节点分析法求解其余部分。

> **举例**：若一个复杂的多级放大电路中，已知第一级输出端的电压，则可将整个后级电路替换为一个电压源，从而独立分析前级电路，大幅降低计算复杂度。

### 2.4 替代定理与等效变换的区别

初学者容易混淆“替代定理”与“等效变换”两个概念：

| 对比项 | 替代定理 | 等效变换 |
|:---:|:---|:---|
| **适用条件** | 需已知被替代支路的电压或电流 | 不需知道具体数值，只需电路结构 |
| **替代/等效对象** | 仅替换**一条支路** | 可替换**整个二端网络** |
| **是否保持全部电路变量** | 替代后，**被替代支路本身的变量消失**，其余部分保持不变 | 等效变换前后，**外部端口电压电流不变**，但内部变量可以改变 |
| **典型例子** | 已知 $I=2A$ 的支路 → 用 $2A$ 电流源替换 | 戴维宁定理：有源二端网络 → 电压源串联电阻 |


## 3. 戴维宁定理与诺顿定理

### 3.1 二端网络的基本概念

在介绍戴维宁定理之前，先明确二端网络的概念：

- **二端网络（单口网络）** ：具有两个出线端的部分电路。
- **无源二端网络**：内部不含独立电源的二端网络，仅含电阻和受控源。
- **有源二端网络**：内部含有独立电源的二端网络。

### 3.2 无源二端网络的等效电阻（$R_{eq}$）

对于**不含独立源、仅含线性电阻**的无源二端网络，其等效电阻可通过电阻的串并联、Y-Δ变换等方法直接求得。

但对于**含有受控源**的无源二端网络，不能直接使用串并联公式，需采用**外施电源法**（也称外加激励法）：

1. 在端口处外施一个电源（电压源 $u$ 或电流源 $i$）。
2. 利用 KCL/KVL 或节点/回路分析法，建立端口电压 $u$ 与端口电流 $i$ 的关系方程。
3. 计算等效电阻：
   $$
   R_{eq} = \frac{u}{i}
   $$
   （注意：$u$ 和 $i$ 相对于端口需为一致参考方向，通常取关联参考方向。）

> **为什么受控源不能直接用串并联？** 因为受控源的输出电压/电流取决于控制量，而控制量又与端口电压/电流相关，这种“双向耦合”关系使得等效电阻不能简单地用串并联来化简，必须通过列方程的方法求出 $u/i$ 的比值。

### 3.3 戴维宁定理

#### 定理内容

**戴维宁定理**：任何一个由线性电阻、线性受控源和独立源构成的**有源线性一端口网络**，对外电路来说，总可以用一个**电压源**和**电阻**串联的电路来等效替代。
- **电压源的电压** $U_{oc}$：等于该一端口网络的**开路电压**（将外电路断开后，端口处的电压）。
- **串联电阻** $R_{eq}$：等于该一端口网络内部**所有独立源置零**（电压源短路、电流源开路）后，从端口看进去的**等效电阻**（也叫输入电阻）。

戴维宁定理的意义在于：无论有源二端网络内部结构多么复杂，对于外电路而言，其电学行为完全可以由**两个参数**——$U_{oc}$ 和 $R_{eq}$——完整描述。这极大简化了对复杂电路中某一条特定支路的分析。

#### 定理的物理理解

戴维宁定理的核心思想基于线性电路中二端网络端口电压与电流的线性关系。对于任意一个有源线性二端网络，其端口的 $u$-$i$ 关系是一条直线，可表示为：

$$
u = U_{oc} - R_{eq} \cdot i
$$

这正是电压源串联电阻的伏安关系！$U_{oc}$ 是 $i=0$ 时的电压（开路电压），$-R_{eq}$ 是直线的斜率（等效电阻）。因此，从外部端口看，任何有源线性二端网络的伏安特性都与一个电压源串联电阻的组合完全相同，这就是戴维宁定理的物理本质。

### 3.4 诺顿定理

**诺顿定理**：任何一个线性有源一端口网络，对外电路来说，可以用一个**电流源**和**电阻**并联的电路来等效替代。

- **电流源的电流** $I_{sc}$：等于该一端口网络的**短路电流**（将端口短路后，流经端口的电流）。
- **并联电阻** $R_{eq}$：与戴维宁等效电阻相同，为所有独立源置零后的端口等效电阻。

#### 戴维宁与诺顿的相互转换

两者描述的是同一个有源二端网络，只是等效形式不同，它们之间可通过**电源等效变换**互相转换：

$$
U_{oc} = I_{sc} \cdot R_{eq} \quad \text{或} \quad R_{eq} = \frac{U_{oc}}{I_{sc}}
$$

这一关系提供了求解等效电阻 $R_{eq}$ 的又一种方法——**开路短路法**：分别求出开路电压 $U_{oc}$ 和短路电流 $I_{sc}$，直接相除即得 $R_{eq}$。

#### 特殊情况的处理

| 情况 | 处理方式 |
|:---|:---|
| $R_{eq}=0$（理想电压源特性） | 只能等效为戴维宁电路，诺顿电路不存在（并联电导无穷大） |
| $R_{eq}=\infty$（理想电流源特性） | 只能等效为诺顿电路，戴维宁电路不存在（串联电阻无穷大） |
| $U_{oc}=0$ 且 $I_{sc}=0$ | 等效电阻可能存在，需用外施电源法求解 |

### 3.5 戴维宁/诺顿定理的解题步骤

1. **断开外电路**：将待求支路从端口处断开，确定需要等效的二端网络。
2. **求开路电压 $U_{oc}$**：计算断开后的端口电压。
   - 可使用节点电压法、回路电流法或叠加定理等。
   - 注意：若受控源的控制变量在被断开的支路中，开路时该支路电流为0，控制量可能随之消失，需特殊处理。
3. **求等效电阻 $R_{eq}$**：
   - **方法一（外施电源法）**：将二端网络内所有独立源置零（电压源短路、电流源开路），**受控源保留**。在端口外施电压源 $u$，求端口电流 $i$，则 $R_{eq} = u/i$。
   - **方法二（开路短路法）**：先求 $U_{oc}$，再求端口短路时的短路电流 $I_{sc}$，则 $R_{eq} = U_{oc}/I_{sc}$。
   - **方法三（串并联化简）**：若二端网络内部**无受控源**，置零独立源后直接用电阻串并联及Y-Δ变换化简。
4. **构建等效电路**：将 $U_{oc}$ 和 $R_{eq}$ 串联（戴维宁），或将 $I_{sc}$ 和 $R_{eq}$ 并联（诺顿），再接回原待求支路。
5. **求解待求支路变量**：在简化的等效电路中计算所需电压或电流。

### 3.6 含受控源电路的处理要点

受控源的处理是戴维宁定理中的核心难点。**关键原则**：求 $R_{eq}$ 时，独立源必须置零，但**受控源必须保留**，因为受控源的输出由控制量决定，其作用与电路结构耦合在一起，不能简单地“置零”。

**控制变量转移**：若受控源的控制变量在被等效电路**之外**（即在被断开的支路中），则需将控制变量用端口电压或电流来表示，否则等效后控制关系会丢失，导致错误。

**外施电源法**是最通用的方法：无论电路中有何种受控源，在端口外施电源，列方程求出 $u/i$ 的比值即可得到 $R_{eq}$。

### 3.7 典型题型处理

| 题型 | 处理方法 |
|:---|:---|
| **含受控源求 $R_{eq}$** | 必须用外施电源法或开路短路法。独立源置零，受控源保留，列方程求 $u/i$ 比值。 |
| **控制变量在外部** | 需先将控制变量**转移**到端口电压或电流上，否则控制关系断裂。 |
| **求某支路变量** | 将该支路断开，剩余部分化为戴维宁等效，再接回该支路求解。 |
| **负电阻情况** | 含受控源电路的等效电阻可能为负值，如实记录即可，不必惊慌。负电阻意味着该二端网络在有源状态下向外提供功率。 |
| **无受控源的简单电路** | 可直接用串并联和Y-Δ变换求 $R_{eq}$。 |
| **求最大功率传输** | 先将有源二端网络化为戴维宁等效电路，负载电阻 $R_L = R_{eq}$ 时获得最大功率，$P_{max} = U_{oc}^2/(4R_{eq})$。 |


## 4. 解题方法总结

### 4.1 电路定理选择指南

| 电路情况 | 推荐使用的定理/方法 |
|:---|:---|
| 多电源线性电路，求某处电压或电流 | **叠加定理** |
| 已知某支路电压或电流，简化电路结构 | **替代定理** |
| 只关心某一条支路的变量，其余部分复杂 | **戴维宁定理**（将剩余部分化简为电压源+电阻） |
| 需要将复杂二端网络简化为最简形式 | **戴维宁/诺顿定理** |
| 求含受控源二端网络的输入电阻 | **外施电源法** 或 **开路短路法** |
| 分析T型网络、梯形网络 | **齐次定理** |
| 求负载获得的最大功率 | **戴维宁定理 + 最大功率传输定理** |

### 4.2 常见陷阱与检查点

- [ ] **叠加定理中受控源是否保留？** —— 必须保留在各分电路中，不能置零。
- [ ] **戴维宁求 $R_{eq}$ 时受控源是否保留？** —— 必须保留，独立源置零。
- [ ] **参考方向是否一致？** —— 叠加求和时注意正负号；外施电源法求 $R_{eq}$ 时 $u$ 和 $i$ 需取关联参考方向。
- [ ] **功率能否叠加？** —— 绝对不能！需先求总电压或总电流，再算功率。
- [ ] **控制变量是否消失？** —— 开路/短路时，检查受控源控制量是否变为0，若控制量在外部则需转移。
- [ ] **替代定理的耦合关系** —— 被替代部分与剩余部分不能有控制量的交叉耦合。
- [ ] **戴维宁等效电路中电压源的极性** —— 应与开路电压 $U_{oc}$ 的方向一致。
- [ ] **$R_{eq}$ 是否可以为负？** —— 可以，当电路中含有受控源和独立源时，等效电阻可能为负值。
