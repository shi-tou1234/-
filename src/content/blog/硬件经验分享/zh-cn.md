## 1. 总体规划与布局原则

你可能觉得布局嘛，把零件放上去就行了。但布局这东西，决定了后面所有工作的成败。就像盖房子，地基没打好，后面装修得再好也白搭。

### 1.1 功率与信号分离

这条原则必须刻在脑子里：**把强电和弱电物理上分开。**

啥意思呢？就是板子上画一条"楚河汉界"，一边是大电流的功率部分，一边是小信号的逻辑部分。

* **强电区**：电池接口、电机驱动模块（DCDC buck 转换器，比如 SCT2450STER 这种）
* **弱电区**：单片机（咱们的 STC32G144K246）、降压稳压模块（5V→3.3V）
* **传感器接口区**：摄像头、编码器、舵机接口

为什么？道理其实很简单。电机转动的时候会有很大的电流波动，会在板子上产生电磁干扰。如果单片机或者传感器的走线离得太近，就会被"带偏"。就像你在 KTV 包厢唱歌，旁边有人开音响，你的声音就被盖住了。

### 1.2 布局顺序

很多人一拿到原理图就开始拉线，这其实是大忌。正确的顺序是这样：

1. **元器件布局（Placement）**：先把所有元器件摆好位置
  * MCU 放在板子正中央 ── 它是大脑，应该在核心位置
  * 电源稳压芯片靠近电池输入口 ── 电力的"第一站"
  * 电机驱动芯片放在远离 CPU 的边缘 ── 让它去角落自己折腾
2. **电源线铺设（Power）**：先画电源线，不要用自动走线。相信我，自动走线出来的电源线，十个有九个不能用
3. **地平面铺铜（GND Plane）**：整板铺地，检查有没有孤立的"死铜"

### 1.3 电流方向意识

这一条可能很多新手意识不到。**安放元器件时，要顺着电流的方向来，不是想怎么摆就怎么摆。**

最好的情况是，电流从板子一端进，从另一端出，一路顺畅地流过去。比如从左到右，干干净净的。不要板子上的线绕来绕去，像迷宫一样。

道理也很直白：电流走弯路 = 导线变长 = 电阻变大 = 压降变大 + 更容易收到干扰。尤其是大电流的路径，绕一圈可能就是灾难。

我个人的习惯是：在画板子之前，先拿笔在纸上画一下电流的"主干道"怎么走，主控在哪，电源在哪，驱动在哪。画明白了再开 EDA。

## 2. 电源系统设计

电源这东西，说难不难，说简单也简单。但有一点必须承认：**电源是所有电路的基础。** 电源不稳，后面什么都白做。

### 2.1 电源架构一览

咱们板子的供电架构是这样的：

    电池 (XT60接口)
      → HY19P03B (PMOS 防反接)     ← 防止电池接反烧板子
      → 电源开关 (SS-12D02)         ← 总开关
      → SCT2450STER (DCDC降压)      ← 把电池电压降到需要的水平
      → RT9013-33 (LDO 3.3V)        ← 给主控提供 3.3V
      → SY6280AAC (限流开关)         ← 对外输出保护

电脑上也有好几个不同的电压网络：

| 电源网络 | 电压  | 主要用途 |
| --- | --- | --- |
| VCCBAT | 电池电压 | 电机驱动、舵机 |
| VCC5V | 5V  | MCU 部分外设 |
| VCC3V3 | 3.3V | MCU I/O、逻辑电路 |
| VCC5VEXT | 5V（外供） | 外部设备供电 |
| VCCCAM | 3.3V | 摄像头供电 |

### 2.2 电源完整性

什么叫电源完整性？说人话就是：**你希望芯片吃到的电是干净的、稳定的。**

但实际情况呢？电机一转，电流突然被抽走一大块，电源线上就会出现"纹波"——就像平静的水面被扔了块石头，泛起一圈圈波纹。这个纹波如果传到传感器或者运放上，就会直接影响数据。

几个必须记住的点：

| 要点  | 怎么理解 |
| --- | --- |
| **独立供电** | 传感器对电压特别敏感，越敏感的器件越要给它单独一路电 |
| **摄像头和运放** | 摄像头工作时会有负载，产生纹波；如果运放和它共用电源，纹波会影响运放，导致运放输出不准 |
| **去耦电容** | MCU 每个 VCC 引脚旁边都要放一个 0.1μF 的陶瓷电容，距离引脚不超过 3mm —— 越近越好，远了就没用了 |
| **DCDC 布局** | 输出电容要紧挨着芯片的 IN 和 GND 引脚，连接线短而粗，回路面积要小。输入端电容先大后小，输出端电容先小后大 |

去耦电容的原理其实挺有意思的：它相当于一个"小水库"——当电路突然需要大量电流的时候，电容先顶上，而不是直接从电源线上抽。这样电源线上的波动就被平滑掉了。

### 2.3 线宽与电流

这是一个经常被忽视但极其重要的问题。**导线是有电阻的，电流大了会发热，甚至烧断。**

| 走线类型 | 建议线宽 | 原因  |
| --- | --- | --- |
| 主动力线（电池→电机驱动） | **80-100mil** | 电机启动瞬间 2-5A，太细就冒烟了 |
| 主电源线（电池→稳压模块） | **40-60mil** | 也有不小的电流经过 |
| 逻辑电源线（稳压→单片机） | **20-30mil** | 电流小了，但也不能太细 |
| 普通信号线 | **10-12mil** | 信号电流极小，细点完全没问题 |
| 5V 连线 | ≥**25mil** | —   |
| 密集型 MCU 引脚 | **8mil** | 引脚太密走不了粗的，而且细线可以减少对其他信号的干扰 |

一个简单的换算关系记一下：**10mil ≈ 0.254mm**。所以电机驱动的线，至少要 2mm 以上宽才放心。

如果实在空间不够，也有一个土办法：在线上去掉阻焊层（就是露铜），然后焊接的时候再补一层锡上去，相当于把线加厚了，也能过更大的电流。

## 3. 元器件选型与使用

### 3.1 电容

电容这东西，看着不起眼，但用错了是会"炸"的——不是开玩笑。

**极性**

* 插入式的（电解电容）：长脚是正极，短脚是负极。板子上会标正负极，千万别焊反
* 贴片的：一半有黑线的那边是负极
* **接反了会炸**。是真的会"嘭"的一声爆开，所以上电前最好检查一下

**额定电压**

选电容的时候，额定电压至少是实际电压的**1.5倍**。比如电路是 5V，那就选耐压 10V 的电容。为什么？因为电压是有波动的，如果刚好卡在边界上，一个浪涌过来电容就坏了。

**ESR 的悖论**

电容内部其实有一个等效串联电阻，简称 ESR。这个值越低，电容的效率越高、发热越小。但有意思的是，ESR 太低的话，滤波能力反而变差了——因为它没法有效消耗高频噪声。

所以这里其实有一个**取舍**：ESR 不是越低越好，要看具体用在哪。电源输入端需要低 ESR，去耦电容则需要适中。

**纹波**

纹波就是直流电压上叠加的交流成分。你可以想象成你想喝一杯白开水，但水面上一直在晃。这个"晃"就是纹波。纹波大了，轻则信号不准，重则直接损坏元器件。

**封装大小**

后来几次改版的经验告诉我们：电容尽量用 **0603 封装**的贴片。比 0805 小，但比 0402 好焊接，是比较折中的选择。

### 3.2 MOS 管

MOS 管有三个脚：**源极(S)**、**栅极(G)**、**漏极(D)**。栅极控制通断，有点像水龙头的手柄。

MOS 管分两种：**NMOS** 和 **PMOS**。怎么记？

我是这么记的：PMOS 的 P 代表 Positive（正的），电流从源极流到漏极，是"顺势而为"。NMOS 的 N 代表 Negative（负的），电流从漏极流到源极，是"逆流而上"。

* NMOS：VGS 大于阈值时导通，电流 D→S
* PMOS：VGS 小于阈值时导通，电流 S→D

咱们板上用到的：

* **AO3400**（NMOS）─ 通用开关
* **S8050**（NPN 三极管）─ 激光管驱动
* **HY19P03B**（PMOS）─ 电池防反接

### 3.3 二极管

有黑线的那一头是负极。贴片 LED 是特例，单看外观看不出来，需要拿万用表量一下才确定。

板上用的二极管：

* **SS54 / SS54F** ─ 肖特基二极管，用在 DCDC 电路中整流
* **BZT52C16** ─ 稳压管

### 3.4 电感

咱们板上用的是 **YSPI0724-4R7M**，4.7μH 的功率电感，用在 DCDC 降压电路（SCT2450STER）的输出端。

电感的作用简单说就是"稳流"——当电流突然变化的时候，电感会"抗拒"这种变化，让电流变化得平缓一些。在 DCDC 电路里，电感和电容配合，把脉冲状的电压变成平滑的直流。

**👉 顺便说一句，后面会讲到，测试 DCDC 电源好不好，量电感两端对地的电压就知道了。**

## 4. PCB 布线规范

### 4.1 走线的基本规则

| 规则  | 怎么理解 |
| --- | --- |
| **不要走直角** | 转弯必须是 135° 钝角。直角会在高频信号下产生信号反射，就像光线打到镜子上一样弹回来，然后和原始信号打架 |
| **不要自动走线** | 特别是电源线，自动走出来的永远是大弯大绕的，绝对不行 |
| **设计规则改一下** | 默认的线宽间距可能是 6mil，太容易串扰了。统一改成 8mil 以上 |
| **走线越短越好** | 接口下面是可以走线的，没必要绕一大圈 |
| **先画电源线** | 电源线是板子的"主动脉"，必须先安排好 |

说真的，90° 直角线是很多新手最容易犯的错误。画完板子后建议用 DRC 检查一下，把所有的直角都标出来，一个一个改。

### 4.2 过孔：并不是越多越好

很多人以为，过孔多代表接地好。不一定。

**过孔的问题：**

第一，每个过孔都有寄生电感和寄生电容。对高频信号来说，每穿过一个过孔就像过了一道关卡，信号质量会下降。穿来穿去的，信号反射越来越严重，这叫"残桩效应"。

第二，过孔密集的地方，PCB 厂商在钻孔时容易出问题。如果一排过孔挨得太近，钻头打下去的时候板子容易变形，良品率下降。去年电赛有一队无刷电机驱动板就是吃过这个亏——生产出错，工期延误，比赛时候没拿到板子，直接凉了。

第三，大量过孔会导致板子应力集中，容易断裂。尤其是靠近板边的位置，过孔一多掰一下就断了。

**所以原则就一个：过孔能少就少。** 不要穿下去又穿上来，没必要。

当然，铺铜区域的地过孔该打还是要打的，连接上下两层地平面。只是别到处乱打。

### 4.3 铺铜

铺铜就是把 PCB 上空白的区域全填上铜，连接到 GND。这不是为了好看，有几个关键作用：

* **屏蔽**：相当于给板子穿了一件"防弹衣"，电机产生的电磁噪声被铜皮挡住，不会干扰到敏感的电路
* **散热**：大面积铜皮就像散热片，帮助芯片散热
* **回流路径**：信号电流需要一个回路，大面积地平面提供了最短的回流路径

**铺铜的注意事项：**

* 全部线画完再铺铜
* 顶层和底层都铺 GND
* 铺完后检查有没有"死铜"（没有连接到地网的孤立铜皮），如果有就多用过孔连上
* 铺铜区域尽量别打过孔（第三版修改意见特意提到要"铺铜的过孔去掉"）

### 4.4 走线禁区

以下这些地方，不管多忙多赶，都**不要**走线：

* **摄像头排线和编码器线**：严禁从电机驱动芯片正下方穿过。电机驱动芯片是大电流、强干扰的地方，信号线从下面走等于主动往干扰堆里钻
* **PCLK（像素时钟）**：摄像头的高频时钟线，千万别靠近电机的正负极线。高频信号加上大电流，干扰直接拉满
* **晶振电路下面**：晶振下面是禁区，不能放元器件，也不能走线。晶振电路做包地的话要打大量地孔
* **芯片底下**：尽量不打过孔。因为芯片焊接时，焊锡可能会流进过孔，造成虚焊或者短路

### 4.5 晶振电路布局

晶振是 MCU 的心跳，它的布局直接影响整个系统的稳定性：

* 晶振尽量靠近芯片引脚，越近越好
* 走线短且直，别绕弯
* 晶振的正下方各层都铺 GND，不要放任何元器件和走线
* 晶振周围做包地处理，打地孔

## 5. 接口与连接器设计

### 5.1 各接口规格

根据多次改版的总结，咱们板上接口的规格如下：

| 接口类型 | 规格  | 用途  |
| --- | --- | --- |
| **电源接口** | XT60PW-M | 电池主电源输入 |
| **编码器接口** | **GH1.25 6Pin** | 编码器连接 |
| **激光接口** | **XH2.45 2Pin** | 紫外激光（注意正负极，有一边带锁扣的才是正面） |
| **有刷电机** | 保留一个接口 | 电机驱动 |
| **舵机** | 保留一个接口 | 转向伺服 |
| **屏幕接口** | **2×8P 排母** | 屏幕连接（逐飞的屏幕有的双排有的单排，2×8 通吃） |
| **下载/调试** | USB Type-C | STC32G 自带 USB，不用额外 CH340 |
| **电源端子** | KF301-5.0-2P（2个） | 外接电源 |

### 5.2 接口设计的经验之谈

**电源开关要重新画**

这个在第一版修改的时候就被指出了——原来用的开关是普通的轻触按键开关，电流额定值只有 50mA，而且弹片容易氧化导致接触不良。后面换成了**自锁式红色开关（SS-12D02-VG4）**，额定电流 3A，带自锁功能，按下即锁定导通状态，不会因为震动而断开，适合作为整板的电源总开关。

**加高排母**

屏幕和核心板的位置是重叠的，如果直接用普通排母，屏幕会压在主板上。需要用加高排母把屏幕"架起来"，让它和主板错层。就像把一层楼板抬高，下面还能走人。

**激光接口用立式，别用卧贴**

这是我个人的惨痛教训。卧贴的接口，线从侧面插进去，如果下面有其他元器件，线就会被挡住。轻则难插拔，重则根本插不进去。立式的接口，线朝上走，问题自然就解决了。

**接口间距留够**

这一点也被多次提到——接口之间靠太近，插拔的时候手指都伸不进去。尤其是排针排母这种需要大力按压的，间距不够真的很痛苦。

**开关都放一起**

板子上如果有多个开关，尽量放在同一个区域。这样操作的时候一目了然，不用到处找开关在哪。

### 5.3 接线颜色约定

这是老规矩了，写在前面：

| 颜色  | 含义  |
| --- | --- |
| **黑色** | GND（地） |
| **红色** | VCC（电源正） |
| **橙色** | PWM（信号） |

**永远按这个颜色来。** 哪天你抓了一把线，红黑橙一看就知道哪根是什么线。要是随便用，调试的时候每根线都要用万用表量一遍，心态直接炸裂。

## 6. STC32G144K246 核心板专项

咱们用的是 **STC32G144K246**，这是一颗 8051 架构增强型的芯片。144 个引脚，自带 USB 模块，最大优势就是**不用外部下载器**，一根 USB 线就能烧录和调试。

### 6.1 和 TC264 最大的区别

之前有人用过英飞凌的 TC264（TriCore 架构，32 位高性能），想直接往 STC 上搬代码。坦率讲，这条路走不通。

几个核心差异：

| 差异  | TC264 | STC32G |
| --- | --- | --- |
| 架构  | TriCore（多核） | 8051 增强（单核） |
| 主频  | 高   | 约 35MHz |
| 浮点运算 | 快   | 较弱  |
| 下载方式 | 需要下载器 | USB 直连 |

所以移植算法的时候有个关键点：**多用整数运算（移位操作），少用浮点数。** STC32G 做复杂的浮点运算容易卡顿，毕竟架构摆在那。

### 6.2 USB 接口布线的生死线

STC32G 自带 USB 模块，但这对布线提出了要求。你不能随便把 D+ 和 D- 拉出去就完事了。

**具体要求：**

* 必须引出 **D+ (P3.0/P1.2)** 和 **D- (P3.1/P1.3)**
* D+ 和 D- 走线**严禁跨越强电流区**（驱动板附近尤其不能走）
* 如果不遵守这条，USB 通讯会极其不稳定——有时候能下载有时候不行，有时候甚至可能烧坏电脑的 USB 口

这个真不是吓人。USB 是差分信号，抗干扰能力虽然比单端信号强，但也不是无敌的。如果高速信号线旁边有大电流跳变，信号质量会严重劣化。

### 6.3 144 个引脚的"全连接"要求

144 个引脚的芯片，电源引脚也不少。**所有 VCC 和 GND 引脚必须全部连接到电源网络。**

有的人可能觉得"反正电源都是通的，接一对就够了吧"。不行。每一个电源引脚都是设计好的，不接意味着芯片内部某些模块的供电路径变长、电阻变大，工作不稳定。

另外，芯片底部有一个散热焊盘，必须大量打孔接到地平面。散热焊盘不光是散热用的，也是提供低阻抗接地路径的一部分。芯片跑图像算法的时候发热量不小，散热做不好会影响性能和寿命。

### 6.4 模拟地和数字地的隔离

如果用了摄像头（咱们确实用了），就涉及模拟地和数字地的问题。

摄像头产生的是模拟信号，但 MCU 处理的是数字信号。模拟电路对噪声敏感，数字电路本身会产生噪声。如果两者的地直接连在一起，数字噪声就窜进模拟部分了。

**解决方法：** 模拟地（AGND）和数字地（DGND）在靠近芯片的地方通过一个**磁珠（Ferrite Bead）**连接。

磁珠的作用是：直流可以顺利通过，但高频噪声被"吃掉"了——变成一个微小的热量散掉。这样地平面的电位还是相等的，但高频噪声过不去。

效果：显示器画面上的纹波和噪点会明显减少。

### 6.5 冷启动与复位

这是个很容易被新手忽视的问题。

STC32G 有一个"冷启动下载"的逻辑。什么意思呢？就是 MCU 在上电的瞬间检测下载信号，如果检测到了就进入下载模式。

所以：

* **正常情况下**：代码没写死的时候，STC-ISP 软件发指令就能自动下载，不需要手动断电
* **代码写崩了**：如果代码里死循环把串口给关了（这种情况很常见），软件就没办法和 MCU 通信了。这个时候必须手动按一下复位键，或者断电再上电才能恢复

所以——**PCB 上一定要留一个复位按键。** 如果没放，代码写崩的时候你就只能反复拔插电源线，非常痛苦。

## 7. 抗干扰与信号完整性

智能车是一个典型的"大电流 + 小信号"并存的环境。电机转动的时候，整个板子都在电磁干扰的"风暴"中。怎么让脆弱的信号在风暴中生存下来？核心思想就是分而治之。

### 7.1 分层策略

| 层面  | 具体做法 | 一句话解释 |
| --- | --- | --- |
| **布局** | 功率电路和逻辑电路物理分开 | 别让大电流和小信号挤在一起 |
| **布线** | 信号线远离功率线 | 物理上隔开就是最好的屏蔽 |
| **接地** | 整板铺铜 | 给信号一个干净的回流路径 |
| **去耦** | 每对 VCC/GND 旁边放电容 | 随时准备给芯片"续杯" |
| **隔离** | AGND 和 DGND 用磁珠连 | 让数字噪声别窜到模拟部分 |

### 7.2 去耦电容怎么放

去耦电容这东西，放了不一定能看出来效果，但不放一定出事。

几个关键点：

* **每个 VCC 引脚旁边都要放一个 0.1μF 的陶瓷电容**
* **距离引脚不超过 3mm**——远了就等于没放
* 除了去耦电容，还要在芯片附近放一个 10μF 的电解电容或钽电容，作为"大水库"

可以这么理解：10μF 的是蓄水池，存大量水；0.1μF 的是水龙头旁边的小杯子，随时可以快速取水。芯片需要电流的瞬间，小杯子先顶上，蓄水池的水再慢慢流过来。

### 7.3 激光管驱动方案

**这是一个绝对不能犯的错误：不要直接把激光管接在 MCU 引脚上。**

为什么？两个原因：

1. **电流不够**：MCU 的 GPIO 引脚最多输出几毫安到十几毫安，但激光管工作需要几十甚至上百毫安的电流
2. **反向电压**：激光管关闭的时候会产生反向电压，可能把 MCU 引脚烧掉

**正确的做法**：用一个 **8050 三极管**做开关，MCU 引脚控制三极管的基极，三极管的集电极驱动激光管。

    MCU 引脚 → 电阻 → 8050 基极
                       8050 集电极 → 激光管
                       8050 发射极 → GND

这样 MCU 只需要提供很小的基极电流，大部分电流从电源直接走三极管到激光管。

### 7.4 EDA 工具操作提示

* 我们用的是嘉立创 EDA（标准版/专业版）。如果需要从 Altium Designer 文件导入，操作步骤是：打开嘉立创 EDA，点击「文件」→「导入」→「Altium Designer」，选择 .SchDoc / .PcbDoc 文件，在弹出的对话框中确认网络和元器件对应关系，点击确定即可。如果导入后有元器件丢失，一般是因为封装库路径不对，需要手动在立创商城里搜索对应的封装并替换
* 嘉立创专业版现在已经没有螺丝孔的封装了，可以在 PCB 上直接放置焊盘代替

## 8. 焊接与调试

这一章是新加的，专门说说拿到板子之后怎么焊、怎么测。

在网上经常能看到的悲剧是：板子画好了，打样回来了，元器件都到了，结果焊上去冒烟了。很多时候不是设计的问题，是焊接和调试的姿势不对。

### 8.1 焊接顺序

**原则就一句话：先焊贴片，再焊直插。**

很多人拿到板子习惯先把排针、排母、插座这些大的东西焊上去。结果发现电阻电容没地方放——被排针挡住了。这个顺序其实是大忌。

**为什么是这个顺序？**

贴片元件（电阻、电容、芯片这些）是直接贴在 PCB 表面的，它们的焊盘都在同一层。直插元件（排针、排母、插座、电解电容这些）是需要穿过 PCB 孔洞的，从背面穿到正面。

如果你先把排针焊上去了，它就立在板子上，你要焊贴片的时候，烙铁头可能就被排针挡住了，够不到里面的焊盘。反过来，如果你先把贴片焊好，板子是平的，烙铁随便动。

**建议的焊接顺序：**

    第一步：先焊最小的贴片元件
      → 电阻、电容（0603/0805 封装）
      → 二极管、LED
      → 小封装 IC（SOT-23、SOP-8 等）
    
    第二步：焊大一点的贴片
      → SOP-28（像 AL422B）、ESOP-8（SCT2450STER）
      → STC32G144K246（QFP 封装，144 脚）
    
    第三步：焊直插元件
      → 排针、排母
      → 电解电容、电感
      → 插座（XT60、KF301 等）
      → 开关、按键

当然，有个例外：如果你手上有那种很矮的贴片元件，旁边有很高的直插元件已经是焊好的，可以先焊直插、反过来放，利用高度差来避免干涉。但对新手来说，还是按上面的顺序最稳妥。

### 8.2 焊接的一些小技巧

**锡量控制**

贴片电阻电容的锡不用太多，薄薄一层就够了。锡太多容易连锡（短路），也容易把元件"顶"起来。

**助焊剂是你的好朋友**

特别是焊那种引脚密集的芯片（144 脚的 STC32G），助焊剂可以帮助焊锡流动，防止连锡和虚焊。焊完用洗板水或者酒精把残留擦掉。

**先固定再焊接**

焊多脚芯片的时候，先把对角两个引脚焊上固定，对齐了再焊剩下的。不然很容易焊上去了才发现芯片歪了。

**检查焊接质量**

焊完之后用放大镜或者手机相机放大看一下，有没有连锡、虚焊、焊锡不够饱满的地方。特别是芯片引脚之间的连锡，眼睛可能看不出来。

### 8.3 上电前的检查

**这一步真的不能跳过，每年至少有一块板子是因为没检查直接上电然后冒烟的。**

要在上电之前做的检查：

1. **万用表测电源对地是否短路**：用蜂鸣档，量 VCC 和 GND 之间。如果滴滴响，说明短路了，查清楚再接电。这是最基础也是最关键的检查
  
2. **检查电源正负有没有接反**：对照原理图，确认电源输入的正负极和板子上的标称一致
  
3. **目视检查有无连锡**：特别是芯片引脚之间、排针的相邻焊盘之间
  
4. **检查所有极性元件方向**：电解电容、二极管、LED 的正负极有没有焊反
  

### 8.4 上电调试流程

板子焊好检查完了，可以上电了。但不要一上来就接电池或者满电压，**一步一步来**：

1. **先不上芯片，只焊电源部分**：如果板子上有稳压芯片（RT9013-33、SCT2450STER 这些），先把电源部分的元件焊好，测量输出电压对不对。万用表打在电压档，测输出电感和地之间的电压。DCDC 电路正常工作时，电感两端对地应该有稳定的电压输出，而且没有明显波动
  
2. **确认电压正常再上芯片**：如果电源输出不稳，或者电压不对，第一时间断电检查，千万别先上芯片。电源炸了还能换，芯片炸了就贵了
  
3. **上电后先摸一下芯片的温度**：如果芯片发烫（不是微热是烫手那种），很可能有短路或者接错了，立刻断电
  
4. **从电源往外一步一步测**：先测 3.3V 正不正常，再测 5V，再测各个外设的供电。从"源头"往"下游"测，哪个节点电压不对，问题就在那一段
  

**测试 DCDC 电源的小技巧：**

为什么让大家量电感两端和地的电压？因为在 DCDC 电路里，电感是储能元件，它正常工作的标志就是两端对地有正确的电压输出。如果电感对地电压不对，要么是 DCDC 芯片有问题，要么是反馈电阻焊错了，要么是电感本身坏了。

## 9. 机械结构与安装

### 9.1 螺丝孔

原理图上不需要画螺丝孔，在 PCB 上直接打孔就行。

**规格**：建议内径 **3mm**，用 **M3 螺丝**安装在车模上。

安装孔的位置要考虑到车模上的孔洞——不一定每个螺丝孔车模上都有对应的位置，所以要提前对齐。

**制作方法：**

一种比较省事的方法：去立创开源中心（oshwhub.com）找一个现成的开源项目（比如泰山派、梁山派、庐山派的板子），它们上面都有螺丝孔的。把工程另存到本地导入嘉立创，右键螺丝孔保存到库中，以后就能直接用了。

### 9.2 板框

**这个一定要加，不能偷懒。** 第一版修改意见里第一条就是"没有板框"。

在 EDA 工具栏里找到板框工具，把板子的外形画出来。不然工厂不知道板子要切成什么形状。

### 9.3 元器件间距

排母和按键焊盘靠太近可能会干涉——排母是插排针用的，如果旁边有按键，插排针的时候可能被按键挡住。所以放置元器件的时候要留出足够的间距。

第三版修改意见也提到了接口间距要变大，这点反复被提起，说明确实是个容易踩坑的地方。

## 10. Checklist：下单前检查清单

**画完板子准备下单之前，对照这个清单一个一个打钩。** 不要嫌麻烦，少烧一块板子抵得过你检查十遍。

* [ ] **电源是否短路**？用万用表蜂鸣档量 VCC 和 GND
* [ ] **线宽改了吗**？确保电机线宽度够了，不是默认的 10mil
* [ ] **丝印清晰吗**？每个排针旁边写上 GND、5V、SDA、SCL。曾经有人没写丝印，焊的时候搞错 GND 和 VCC，板子直接冒烟
* [ ] **激光管驱动电路**：确认激光管是接在三极管或者 MOS 管上，没有直连 MCU
* [ ] **下载口**：留好串口（TX/RX）和地（GND）的排针
* [ ] **复位按键**：有没有放复位按键？
* [ ] **板框**：有没有画板框？
* [ ] **螺丝孔**：有没有加安装孔？
* [ ] **铺铜检查**：整板铺地了没有？有没有死铜（孤岛）？
* [ ] **接口间距**：连接器之间有没有留够插拔空间？
* [ ] **所有 VCC/GND 引脚**：MCU 的每个电源引脚都接了吗？
* [ ] **二维码**：板子上加二维码了吗？（第4版意见提到的）

## 11. 各版本修改记录汇总

这些是从各个版本的修改意见里整理出来的，记录了每一次改板中我们发现了什么问题、做了哪些修改。

### 第一版原理图修改

第一版原理图画完后审查发现的问题：

| 修改项 | 详细说明 |
| --- | --- |
| 电源接口开关 | 原来使用的开关过流能力不够，需要重新选型。最终换成了 **SS-12D02-VG4** 自锁开关，额定电流 3A |
| 安装孔 | 原理图上不需要画安装孔的符号，但 PCB 布板时必须在四个角放置螺丝孔，用来固定到车模上 |
| 排母  | 原来用的普通排母高度太低，屏幕和核心板位置重叠后会压到主板。需要换成**加高排母**，让屏幕和主板之间形成错层 |
| 编码器接口 | 编码器需要用 **GH1.25 6Pin** 接口，不能随便用杜邦线，否则容易松动导致编码器读数不准 |
| 激光接口 | 紫外激光模块使用 **XH2.45 2Pin** 接口，这种接口带锁扣，插上后不会松脱。布线时注意正负极不要画反，带锁扣的一侧是正极 |
| 有刷电机 | 板子上要保留一个有刷电机的接口，使用 **KF301-5.0-2P** 端子或者 XT60 接口，保证能过 2-5A 电流 |
| 舵机  | 保留一个舵机接口，接到 VCCSERVO 电源网络 |
| 屏幕接口 | 逐飞的屏幕有双排和单排两种，为了兼容性统一使用 **2×8P 排母**，双排全引出，单排用其中一排 |
| 开关类型 | 原来的开关从轻触按键改为自锁开关，确保下载程序时能可靠断电再上电 |

### 第一版 PCB 布局修改

打样回来的板子审查后需要修改的问题：

| 修改项 | 详细说明 |
| --- | --- |
| 板框  | 原文件完全没画板框的外形轮廓线，工厂不知道怎么切割板子形状。需要在 EDA 工具栏找到「板框」工具，画出整块板子的外形 |
| 螺丝孔 | 在 PCB 上预留安装孔位置，孔径 **3mm**（适配 M3 螺丝）。可以直接从立创开源项目（如泰山派、梁山派的工程）里把螺丝孔封装导入到自己库中 |
| 走线宽度 | MCU 引脚密集区域使用 **8mil** 线宽，既能走通又能减少引脚间干扰。在 EDA 中用 Ctrl+F 搜索所有 10mil 的导线，批量选中后右侧属性栏统一改为 8mil |
| 设计规则 | 如果默认的线距是 6mil，全部改为 ≥8mil。因为对高频信号之间的串扰机制不够清楚，最简单的方法就是拉开距离，间距大了什么干扰都传不过来 |
| 元器件干涉 | 原来的设计中按键的焊盘和排母的位置太近，焊接后排母会挡住按键。需要把按键往旁边移动 2-3mm 避开排母 |
| 过孔使用 | 原设计在信号路径上打了大量过孔，穿来穿去。过孔有寄生电感和电容，对信号质量不好，而且密集过孔会导致 PCB 生产时应力集中、容易断裂。需要优化走线路径，减少不必要的过孔 |
| 激光接口封装 | 原设计用的卧贴接口，线从侧面插出，会和下方的元器件干涉。改成**立式弯针接口**，线缆朝上方走，不再被挡住 |
| 走线路径 | 原设计有些信号线为了"好看"绕了一大圈，实际上接口下面是可以直接走线的。走线原则是「两点之间尽量短」，不必追求视觉上的整齐而绕远路 |

### 第二版修改意见

第二版原理图审查时提出的修改要求：

| 修改项 | 详细说明 |
| --- | --- |
| 电源接口开关 | 重新选型电源开关，原开关过流能力不足，需更换为额定电流更大的型号 |
| 安装孔 | 原理图上不需要画安装孔符号，在 PCB 布板时直接打孔即可，用于固定螺丝 |
| 排母高度 | 屏幕与核心板位置重叠，需更换**加高排母**，使屏幕与主板形成错层，避免挤压 |
| 编码器接口 | 改用 **GH1.25 6Pin** 接口，替代杜邦线，防止松动导致编码器读数异常 |
| 激光接口 | 改用 **XH2.45 2Pin** 接口（带锁扣），注意正负极不要画反，带锁扣一侧为正极 |
| 有刷电机 | 板子上保留一个有刷电机接口 |
| 舵机  | 板子上保留一个舵机接口 |
| 屏幕接口 | 兼容逐飞双排/单排屏幕，统一使用 **2×8P 排母**，双排全引出，单排用其中一排 |
| 开关类型 | 开关需要更换，确保可靠通断 |

### 第三版修改意见

第三版 PCB 打样回来后发现的问题：

| 修改项 | 详细说明 |
| --- | --- |
| 铺铜区域的过孔 | 原来在 GND 铺铜上打了大量过孔，但有一部分过孔是冗余的（同一块铺铜区域上下层已经通过其他过孔连通了）。过多的过孔反而增加了加工难度和成本，把不必要的过孔全部删除，只保留必要的 GND 连接过孔 |
| 接口间距 | 连接器之间的间距原来只有 2mm，手指伸不进去，拔插排线非常困难。所有的接口间距加大到 **5mm 以上**，确保能正常插拔 |
| 开关位置 | 板子上的电源开关和复位按键原来分散在两个角落，操作时需要换手。全部**集中摆放在板子同一侧边缘**，单手可以操作 |

### 第4版修改意见

第四版审查时提出的修改要求：

| 修改项 | 详细说明 |
| --- | --- |
| 电容封装 | 原来部分电容用的是 0805 封装，现在统一改为 **0603 贴片封装**。0603 比 0805 小，能节省板子空间，而且手工焊接的难度也不算太大 |
| 5V 导线宽度 | 检查发现 5V 电源走线部分段落宽度不足 25mil。高亮所有 5V 网络，确保每一段 5V 走线宽度 ≥ **25mil**（从电源输入端到负载端的整条路径上都不能有瓶颈段） |
| 电流方向 | 之前布局时没有考虑电流的流向，导致有些走线在板子上绕了好几圈。以后布局时，先规划电流从输入到输出的**主干路径**，元器件沿着电流方向依次摆放，不让电流走回头路 |
| 个人信息铺铜 | 板子顶层的个人信息文字（版本号、日期、姓名等）原来没有铺铜，容易看不清楚。现在改为**顶层铺铜字符**，铜皮上刻出来的文字更清晰且不会脱落 |
| 二维码 | 在板子空白处添加一个二维码，扫描后可以跳转到设计文档或原理图链接，方便后续调试时快速查阅电路 |

### 重要注意事项 · 优先级速查

| 序号  | 经验  | 优先级 |
| --- | --- | --- |
| 1   | **电源线必须加宽** ─ 电机主路径 80-100mil | 🔴 必须 |
| 2   | **去耦电容必须放** ─ 每个 VCC 引脚旁 0.1μF | 🔴 必须 |
| 3   | **激光管加三极管驱动** ─ 不能直连 MCU | 🔴 必须 |
| 4   | **整板铺铜** ─ 屏蔽电磁噪声 | 🔴 必须 |
| 5   | **焊接先贴片后直插** ─ 避免焊接干涉 | 🔴 必须 |
| 6   | **上电前测短路** ─ VCC 对 GND 蜂鸣档 | 🔴 必须 |
| 7   | **避免直角走线** ─ 用 135° 钝角 | 🟡 重要 |
| 8   | **布局先分区** ─ 强电/弱电分离 | 🟡 重要 |
| 9   | **模拟/数字地磁珠隔离** ─ 减少摄像头噪点 | 🟡 重要 |
| 10  | **减少过孔** ─ 降低寄生效应和生产风险 | 🟡 重要 |
| 11  | **接口间距留足** ─ 考虑插拔空间 | 🟡 重要 |
| 12  | **丝印标注清晰** ─ GND/5V/SDA/SCL 不能少 | 🟢 建议 |
| 13  | **安装孔打 M3** ─ 内径 3mm，匹配车模 | 🟢 建议 |
| 14  | **测 DCDC 量电感** ─ 电感两端对地电压 | 🟢 建议 |
