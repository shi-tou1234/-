# 欢迎来到我的博客！

这是我使用 **Hexo** 搭建在 GitHub Pages 上的第一篇博客文章。

## 关于这个博客

这个博客使用了以下技术栈：

- **Hexo** - 快速、简洁且高效的博客框架
- **GitHub Pages** - 免费的静态网站托管服务
- **GitHub Actions** - 自动化构建和部署

## 开始写作

你可以通过以下命令创建新文章：

```bash
hexo new "文章标题"
```

然后在 `source/_posts/` 目录下编辑生成的 Markdown 文件即可。

## 常用命令

```bash
hexo new "My New Post"   # 创建新文章
hexo generate            # 生成静态文件
hexo server              # 启动本地预览服务
hexo clean               # 清除缓存
```
